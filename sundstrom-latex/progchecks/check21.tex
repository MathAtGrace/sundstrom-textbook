\subsection*{Progress Check \ref{pr:onlyif}}
\begin{enumerate}
\item Whenever a quadrilateral is a square, it is a rectangle, or a quadrilateral is a rectangle whenever it is a square.

\item A quadrilateral is a square only if it is a rectangle.

\item Being a rectangle is necessary for a quadrilateral to be a square.

\item Being a square is sufficient for a quadrilateral to be a rectangle.
\end{enumerate}
