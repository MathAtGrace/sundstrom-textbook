Definition 5.5.5.

Let \(\Lambda\) be a nonempty indexing set and let \(\mathscr{A} = \left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) be an indexed family of sets. The union over \(\mathbf{\mathscr{A}}\) is defined as the set of all elements that are in at least one of sets \(A_\alpha\text{,}\) where \(\alpha \in \Lambda\text{.}\) We write

\begin{equation*} \bigcup_{\alpha \in \Lambda}^{}A_\alpha = \left\{ x \in U \mid \text{ there exists an } \alpha \in \Lambda \text{ with } x \in A_\alpha \right\}\!\text{.} \end{equation*}

The intersection over \(\mathbf{\mathscr{A}}\) is the set of all elements that are in all of the sets \(A_\alpha\) for each \(\alpha \in \Lambda\text{.}\) That is,

\begin{equation*} \bigcap_{\alpha \in \Lambda}^{}A_\alpha = \left\{ x \in U \mid \text{ for all } \alpha \in \Lambda, x \in A_\alpha \right\}\!\text{.} \end{equation*}
