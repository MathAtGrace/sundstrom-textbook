Progress Check 3.1.12. Proving Proposition 3.1.11.

We will use “backward questions” and “forward questions” to help construct a proof for Proposition 3.1.11. So, we might ask, “How do we prove that \(\left( a + b \right) \equiv 2 \pmod 8\text{?}\)” One way to answer this is to use the definition of congruence and state that \(\left( a + b \right) \equiv 2 \pmod 8\) provided that 8 divides \((a + b - 2)\text{.}\)
