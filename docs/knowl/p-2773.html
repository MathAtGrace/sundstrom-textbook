A number system that we have not yet discussed is the set of complex numbers. The complex numbers, \(\mathbb{C}\text{,}\) consist of all numbers of the form \(a + bi\text{,}\) where \(a, b \in \R\) and \(i = \sqrt{-1}\) (or \(i^2 = -1\)). That is,

\begin{equation*} \mathbb{C} = \left\{ a + bi \left| a, b \in \R \text{ and } i = \sqrt{-1} \right. \right\}\!\text{.} \end{equation*}

We can add and multiply complex numbers as follows: If \(a, b, c, d \in \R\text{,}\) then

\begin{align*} \left( a + bi \right) + \left( c + di \right) \amp = \left(a + c \right) + \left(b + d \right)i, \text{ and }\\ \left( a + bi \right) \left( c + di \right) \amp = ac + adi + bci + bdi^2\\ \amp = \left(ac - bd \right) + \left(ad + bc \right)i\text{.} \end{align*}
