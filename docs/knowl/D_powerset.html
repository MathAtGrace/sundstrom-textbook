Definition.

If \(A\) is a subset of a universal set \(U\text{,}\) then the set whose members are all the subsets of \(A\) is called the power set of \(A\text{.}\) We denote the power set of \(A\) by \(\mathcal{P}( A )\) . Symbolically, we write

\begin{equation*} \mathcal{P}( A ) = \left\{ {X \subseteq U \mid X \subseteq A} \right\}\text{.} \end{equation*}

That is, \(X \in \mathcal{P}( A )\) if and only if \(X \subseteq A\text{.}\)
