A 2 by 2 matrix over \(\mathbb{R}\) is a rectangular array of four real numbers arranged in two rows and two columns. We usually write this array inside brackets (or parentheses) as follows:

\begin{equation*} A = \left[ {\begin{array}{*{20}c} a \amp b \\ c \amp d \end{array} } \right]\text{,} \end{equation*}

where \(a\text{,}\) \(b\text{,}\) \(c\text{,}\) and \(d\) are real numbers. The determinant of the 2 by 2 matrix \(A\text{,}\) denoted by \(\det( A )\text{,}\) is defined as

\begin{equation*} \det( A ) = ad - bc\text{.} \end{equation*}
(a)

Calculate the determinant of each of the following matrices:

\begin{equation*} \left[ {\begin{array}{*{20}c} 3 \amp 5 \\ 4 \amp 1 \end{array} } \right],\; \left[ {\begin{array}{*{20}c} 1 \amp 0 \\ 0 \amp 7 \end{array} } \right],\; \text{ and } \left[ {\begin{array}{*{20}c} 3 \amp { - 2} \\ 5 \amp 0 \end{array} } \right]\text{.} \end{equation*}
Answer.
