Exercise 1.

Let \(R_5 = \left\{ {0, 1, 2, 3, 4} \right\}\text{.}\) Define \(f\x R_5 \to R_5\) by \(f( x ) = x^2 + 4 \pmod 5\text{,}\) and define \(g\x R_5 \to R_5\) by \(g( x ) = (x + 1)(x + 4) \pmod 5\text{.}\)

(a)

Calculate \(f( 0 )\text{,}\) \(f( 1 ), f( 2 )\text{,}\) \(f( 3 )\text{,}\) and \(f( 4 )\text{.}\)

(b)

Calculate \(g( 0 )\text{,}\) \(g( 1 ), g( 2 )\text{,}\) \(g( 3 )\text{,}\) and \(g( 4 )\text{.}\)

(c)

Is the function \(f\) equal to the function \(g\text{?}\) Explain.
