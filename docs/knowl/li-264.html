Item 2.

The statement of the Division Algorithm contains the new phrase, “there exist unique integers \(q\) and \(r\) such that \(\ldots \text{.}\)” This means that there is only one pair of integers \(q\) and \(r\) that satisfy both the conditions \(a = bq + r\) and \(0 \leq r \lt b\text{.}\) As we saw in Beginning Activity 1, there are several different ways to write the integer \(a\) in the form \(a = bq + r\text{.}\) However, there is only one way to do this and satisfy the additional condition that \(0 \leq r \lt b\text{.}\)
