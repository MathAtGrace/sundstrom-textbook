Exercise 3.

Let \(f\x \mathbb{Z} \to \mathbb{Z}\) be defined by \(f( m ) = 3 - m\text{.}\)

(a)

Evaluate \(f( { - 7} ), f( { - 3} ), f( 3 ), \text{ and } f( 7 )\text{.}\)

(b)

Determine the set of all of the preimages of 5 and the set of all of the preimages of 4.

(c)

Determine the range of the function \(f\text{.}\)

(d)

This function can be considered a real function since \(\Z \subseteq \R\text{.}\) Sketch a graph of this function. Note: The graph will be an infinite set of points that lie on a line. However, it will not be a line since its domain is not \(\R\) but is \(\Z\text{.}\)
