It is possible to prove the relationship suggested in Progress Check 5.3.5 by proving that each set is a subset of the other set. However, the results in Theorems 5.3.2 and Theorem 5.3.4 can be used to prove other results about set operations. When we do this, we say that we are using the algebra of sets to prove the result. For example, we can start by using one of the basic properties in Theorem 5.3.4 to write

\begin{equation*} \left( {A \cup B} \right) - C = \left( {A \cup B} \right) \cap C^c\text{.} \end{equation*}
