Theorem 5.3.4.

Let \(A\) and \(B\) be subsets of some universal set \(U\text{.}\) Then the following are true:

Basic Properties

\(\left( A^c \right)^c = A\)

\(A - B = A \cap B^c\)

Empty Set and Universal Set

\(A - \emptyset = A\) and \(A - U = \emptyset\)

\({\emptyset}^c = U\) and \(U^c = \emptyset\)

De Morgan's Laws

\(\left ({A \cap B} \right)^c = A^c \cup B^c\)

\(\left ({A \cup B} \right)^c = A^c \cap B^c\)

Subsets and Complements

\(\displaystyle A \subseteq B \text{ if and only if } B^c \subseteq A^c\)

Proof.
