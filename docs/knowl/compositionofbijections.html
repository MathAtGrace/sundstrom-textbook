Theorem 6.42.

Let \(f\x A \to B\) and \(g\x B \to C\) be bijections. Then \(g \circ f\) is a bijection and \(( {g \circ f} )^{ - 1} = f^{ - 1} \circ g^{ - 1}\text{.}\)

Proof.

Let \(f\x A \to B\) and \(g\x B \to C\) be bijections. Then \(f^{ - 1} \x B \to A\) and \(g^{ - 1} \x C \to B\text{.}\) Hence, \(f^{ - 1} \circ g^{ - 1} \x C \to A\text{.}\) Also, by TheoremÂ 6.29, \(g \circ f\x A \to C\) is a bijection, and hence \(( {g \circ f} )^{ - 1} \x C \to A\text{.}\) We will now prove that for each \(z \in C\text{,}\) \(( {g \circ f} )^{ - 1} ( z ) = ( {f^{ - 1} \circ g^{ - 1} } )( z )\text{.}\)

Let \(z \in C\text{.}\) Since the function \(g\) is a surjection, there exists a \(y \in B\) such that

\begin{equation*} g( y ) = z \qquad (1)\text{.} \end{equation*}

Also, since \(f\) is a surjection, there exists an \(x \in A\) such that

\begin{equation*} f( x ) = y \qquad (2)\text{.} \end{equation*}

Now these two equations can be written in terms of the respective inverse functions as

\begin{align*} g^{ - 1} ( z ) \amp = y; \text{ and } \amp \amp (3)\\ f^{ - 1} ( y ) \amp = x \amp \amp (4) \end{align*}

Using equation (3) and equation (4), we see that

\begin{align*} \left( {f^{ - 1} \circ g^{ - 1} } \right)( z ) \amp = f^{ - 1} \left( {g^{ - 1} ( z )} \right) \amp \amp\\ \amp = f^{ - 1} ( y ) \amp \amp\\ \amp = x \amp \amp (5)\text{.} \end{align*}

Using equation (1) and equation (2) again, we see that \(( {g \circ f} )( x ) = z\text{.}\) However, in terms of the inverse function, this means that

\begin{equation*} ( {g \circ f} )^{ - 1} ( z ) = x \qquad (6)\text{.} \end{equation*}

Comparing equation (5) and equation (6), we have shown that for all \(z \in C\text{,}\)

\(( {g \circ f} )^{ - 1} ( z ) = ( {f^{ - 1} \circ g^{ - 1} } )( z )\text{.}\) This proves that \(( {g \circ f} )^{ - 1} = f^{ - 1} \circ g^{ - 1}\text{.}\)
