Progress Check 2.4.10. Multiples of Three.

Definition 2.4.11.

An integer \(n\) is a multiple of 3 provided that there exists an integer \(k\) such that \(n = 3k\text{.}\)

(a)

Write this definition in symbolic form using quantifiers by completing the following:

An integer \(n\) is a multiple of 3 provided that ….

(b)

Give several examples of integers (including negative integers) that are multiples of 3.

(c)

Give several examples of integers (including negative integers) that are not multiples of 3.

(d)

Use the symbolic form of the definition of a multiple of 3 to complete the following sentence: “An integer \(n\) is not a multiple of 3 provided that ….”

(e)

Without using the symbols for quantifiers, complete the following sentence: “An integer \(n\) is not a multiple of 3 provided that ….”
