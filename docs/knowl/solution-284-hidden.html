Since 21 does not divide 40, TheoremÂ 8.24 tells us that the Diophantine equation \(63x + 336y = 40\) has no solutions. Remember that this means there is no ordered pair of integers \(( x, y )\) such that \(63x + 336y = 40\text{.}\) However, if we allow \(x\) and \(y\) to be real numbers, then there are real number solutions. In fact, we can graph the straight line whose equation is \(63x + 336y = 40\) in the Cartesian plane. From the fact that there is no pair of integers \(x, y\) such that \(63x + 336y = 40\text{,}\) we can conclude that there is no point on the graph of this line in which both coordinates are integers.
