If we let \(\R^+\) be the set of positive real numbers, then we have a family of sets indexed by \(\R^+\text{.}\) We will first determine the union of this family of sets. Notice that for each \(\alpha \in \R^+\text{,}\) \(\alpha \in A_\alpha\text{,}\) and if \(y\) is a real number with \(-1 \lt y \leq 0\text{,}\) then \(y \in A_\alpha\text{.}\) Also notice that if \(y \in \R\) and \(y \lt -1\text{,}\) then for each \(\alpha \in \R^+\text{,}\) \(y \notin A_\alpha\text{.}\) With these observations, we conclude that

\begin{equation*} \bigcup_{\alpha \in \R^+}^{}A_\alpha = \left( -1, \infty \right) = \left\{ x \in \R \mid -1 \lt x \right\}\!\text{.} \end{equation*}
