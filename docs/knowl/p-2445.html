Prove that for every \(k \in \mathbb{N}\text{,}\) if \(k \geq M\) and \(\left\{ {M,M + 1, \ldots ,k} \right\} \subseteq T\text{,}\) then \(\left( {k + 1} \right) \in T\text{.}\) That is, prove that if \(P( M ),P( {M + 1} ), \ldots ,P( k )\) are true, then \(P( {k + 1} )\) is true.
