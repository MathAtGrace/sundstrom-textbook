Example 6.6.4. Images and Preimages of Sets.

Let \(f\x \mathbb{R} \to \mathbb{R}\) be defined by \(f( x ) = x^2\text{,}\) for each \(x \in \mathbb{R}\text{.}\) The following results are based on the examples in Beginning Activity 1 and Beginning Activity 2.

  • Let \(A = \left\{ 1, 2, 3, -1 \right\}\text{.}\) Then \(f ( A ) = \left\{ 1, 4, 9 \right\}\text{.}\)

  • Let \(B = \left\{ 1, 9, 15, -1 \right\}\text{.}\) Then \(f^{-1} ( B ) = \left\{ -\sqrt{15}, -3, -1, 1, 3, \sqrt{15} \right\}\text{.}\)

The graphs from Beginning Activity 2 illustrate the following results:

  • If \(T\) is the closed interval \(\left[ 1, 2 \right]\text{,}\) then the image of the set \(T\) is

    \begin{align*} f ( T ) \amp = \left\{ f ( x ) \mid x \in \left[ 1, 2 \right] \right\}\\ \amp = \left[ 1, 4 \right]\text{.} \end{align*}

  • If \(C\) is the closed interval \(\left[ 1, 4 \right]\text{,}\) then the preimage of the set \(C\) is

    \begin{equation*} f^{-1} ( C ) = \left\{ x \in \mathbb{R} \mid f ( x ) \in \left[ 1, 4 \right] \right\} = \left[ -2, -1 \right] \cup \left[ 1, 2 \right]\text{.} \end{equation*}
