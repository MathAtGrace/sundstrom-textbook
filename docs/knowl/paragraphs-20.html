Some comments about the negation.

Although the statement, \(\mynot P\text{,}\) can be read as “It is not the case that \(P\text{,}\)” there are often better ways to say or write this in English. For example, we would usually say (or write):

The negation of the statement, “391 is prime” is “391 is not prime.”
The negation of the statement, “\(12 \lt 9\)” is “\(12 \geq 9\text{.}\)”
