Exercise 3.

For each of the following statements

  • Write the statement as an English sentence that does not use the symbols for quantifiers.

  • Write the negation of the statement in symbolic form in which the negation symbol is not used.

  • Write a useful negation of the statement in an English sentence that does not use the symbols for quantifiers.

(a)

\(\left( {\exists x \in \mathbb{Q}} \right)\left( {x > \sqrt 2 } \right)\text{.}\)

(b)

\(\left( {\forall x \in \mathbb{Q}} \right)\left( {x^2 - 2 \ne 0} \right)\text{.}\)

(c)

\(\left( {\forall x \in \mathbb{Z}} \right)\left( {x\text{ is even or } x\text{ is odd } } \right)\text{.}\)

(d)

\(\left( {\exists x \in \mathbb{Q}} \right)\left( {\sqrt 2 \lt x \lt \sqrt 3 } \right)\text{.}\)

Note: The sentence “\(\sqrt 2 \lt x \lt \sqrt 3\)” is actually a conjuction. It means \(\sqrt 2 \lt x\) and \(x \lt \sqrt 3\text{.}\)
