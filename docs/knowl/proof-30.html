Proof

We will prove that \(A \cap B = B \cap A\text{.}\) Let \(x \in U\text{.}\) Then

\begin{equation*} x \in A \cap B\text{ if and only if } x \in A\text{ and } x \in B \qquad (1)\text{.} \end{equation*}

However, we know that if \(P\) and \(Q\) are statements, then \(P \wedge Q\) is logically equivalent to \(Q \wedge P\text{.}\) Consequently, we can conclude that

\begin{equation*} x \in A\text{ and } x \in B\text{ if and only if } \;x \in B\text{ and } x \in A \qquad (2)\text{.} \end{equation*}

Now we know that

\begin{equation*} x \in B\text{ and } x \in A\text{ if and only if } x \in B \cap A \qquad (3)\text{.} \end{equation*}

This means that we can use (1), (2), and (3) to conclude that

\begin{equation*} x \in A \cap B\text{ if and only if } x \in B \cap A\text{,} \end{equation*}

and, hence, we have proved that \(A \cap B = B \cap A\text{.}\)
