In previous mathematics courses, we have frequently used subsets of the real numbers called intervals. There are some common names and notations for intervals. These are given in the following table, where it is assumed that \(a\) and \(b\) are real numbers and \(a \lt b\text{.}\)

Interval
Notation		 Set Notation Name
\(\left( {a,b} \right) =\) \(\left\{ {x \in \mathbb{R}\left. \right|a \lt x \lt b} \right\}\) Open interval from \(a\) to \(b\)
\(\left[ {a,b} \right] =\) \(\left\{ {x \in \mathbb{R}\left. \right|a \leq x \leq b} \right\}\) Closed interval from \(a\) to \(b\)
\(\left[ {a,b} \right) =\) \(\left\{ {x \in \mathbb{R}\left. \right|a \leq x \lt b} \right\}\) Half-open interval
\(\left( {a,b} \right] =\) \(\left\{ {x \in \mathbb{R}\left. \right|a \lt x \leq b} \right\}\) Half-open interval
\(\left( {a, + \infty } \right) =\) \(\left\{ {\left. {x \in \mathbb{R}} \right|x > a} \right\}\) Open ray
\(\left( { - \infty ,b} \right) =\) \(\left\{ {\left. {x \in \mathbb{R}} \right|x \lt b} \right\}\) Open ray
\(\left[ {a, + \infty } \right) =\) \(\left\{ {\left. {x \in \mathbb{R}} \right|x \geq a} \right\}\) Closed ray
\(\left( { - \infty ,b} \right] =\) \(\left\{ {\left. {x \in \mathbb{R}} \right|x \leq b} \right\}\) Closed ray
