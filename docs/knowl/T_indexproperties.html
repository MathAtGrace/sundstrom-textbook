Theorem 5.5.8.

Let \(\Lambda\) be a nonempty indexing set and let \(\mathscr{A} = \left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) be an indexed family of sets, each of which is a subset of some universal set \(U\text{.}\) Then

  1. For each \(\beta \in \Lambda\text{,}\) \(\bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha \subseteq A_\beta\text{.}\)

  2. For each \(\beta \in \Lambda\text{,}\) \(A_\beta \subseteq \bigcup\limits_{\alpha \in \Lambda}^{}A_\alpha\text{.}\)

  3. \(\displaystyle \left(\:\bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha \right)^c = \bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha}^c\)

  4. \(\displaystyle \left(\:\bigcup\limits_{\alpha \in \Lambda}^{}A_\alpha \right)^c = \bigcap\limits_{\alpha \in \Lambda}^{}A_{\alpha}^c\)

Item 3 and Item 4 are known as De Morgan's Laws.
