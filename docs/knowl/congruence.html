Definition.

Let \(n \in \mathbb{N}\text{.}\) If \(a\) and \(b\) are integers, then we say that \(\boldsymbol{a}\) is congruent to \(\boldsymbol{b}\) modulo \(\boldsymbol{n}\) provided that \(n\) divides \(a - b\text{.}\) A standard notation for this is \(a \equiv b \pmod n\text{.}\) This is read as “\(a\) is congruent to \(b\) modulo \(n\)” or “\(a\) is congruent to \(b\) mod \(n\text{.}\)”
