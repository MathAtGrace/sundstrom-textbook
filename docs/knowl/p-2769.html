When dealing with the power set of \(A\text{,}\) we must always remember that \(\emptyset \subseteq A\) and \(A \subseteq A\text{.}\) For example, if \(A = \left\{ {a,b} \right\}\text{,}\) then the subsets of \(A\) are

\begin{equation} \emptyset,\left\{ a \right\}\!,\left\{ b \right\}\!,\left\{ {a,b} \right\}\!\text{.}\tag{5.1} \end{equation}
