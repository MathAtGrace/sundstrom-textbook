We can write this as

\begin{equation*} \mathcal{P}( A ) = \left\{ {\emptyset,\left\{ a \right\}\!,\left\{ b \right\}\!, \left\{ {a,b} \right\}} \right\}\!\text{.} \end{equation*}

Now let \(B = \left\{ {a, b, c} \right\}\text{.}\) Notice that \(B = A \cup \{ c \}\text{.}\) We can determine the subsets of \(B\) by starting with the subsets of \(A\) in (1). We can form the other subsets of \(B\) by taking the union of each set in (1) with the set \(\{c \}\text{.}\) This gives us the following subsets of \(B\text{.}\)

\begin{equation*} \{ c \},\left\{ a, c \right\}\!,\left\{ b, c \right\}\!,\left\{ {a,b, c} \right\}\! \qquad (2)\text{.} \end{equation*}

So the subsets of \(B\) are those sets in (1) combined with those sets in (2). That is, the subsets of \(B\) are

\begin{equation*} \emptyset,\left\{ a \right\}\!,\left\{ b \right\}\!,\left\{ {a,b} \right\}, \{ c \}, \left\{ a, c \right\}\!,\left\{ b, c \right\}\!,\left\{ {a,b, c} \right\}\! \qquad (3)\text{,} \end{equation*}

which means that

\begin{equation*} \mathcal{P}( B ) = \left\{ {\emptyset,\left\{ a \right\}\!, \left\{ b \right\}\!, \left\{ {a, b} \right\}\!,\left\{ c \right\}\!,\left\{ {a,c} \right\}\!,\left\{ {b,c} \right\}\!,\left\{ {a,b,c} \right\}} \right\}\!\text{.} \end{equation*}
