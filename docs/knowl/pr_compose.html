\(g \x B \to B\) defined by \(g(1) = 3\text{,}\) \(g(2) = 1\text{,}\) and \(g(3) = 2\text{.}\)

Create arrow diagrams for the functions \(f\text{,}\) \(g\text{,}\) \(g \circ f\text{,}\) and \(g \circ g\text{.}\)
