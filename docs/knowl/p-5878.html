Let \(A\) be a set. In Section 5.1, we defined the power set \(\mathcal{P} ( A )\) of \(A\) to be the set of all subsets of \(A\text{.}\) This means that \(X \in \mathcal{P} ( A )\) if and only if \(X \subseteq A\text{.}\) Theorem 5.13 in Section 5.1 states that if a set \(A\) has \(n\) elements, then \(A\) has \(2^n\) subsets or that \(\mathcal{P} ( A )\) has \(2^n\) elements. Using our current notation for cardinality, this means that if \(\text{ card} ( A ) = n\text{,}\) then \(\text{ card} ( \mathcal{P} ( A ) ) = 2^n\text{.}\) (The proof of this theorem was Activity 29.)
