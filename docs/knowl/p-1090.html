In calculus, we define a function \(f\) to be continuous at a real number \(a\) provided that for every \(\varepsilon > 0\text{,}\) there exists a \(\delta > 0\) such that if \(\left| {x - a} \right| \lt \delta\text{,}\) then \(\left| {f\left( x \right) - f\left( a \right)} \right| \lt \varepsilon\text{.}\)
