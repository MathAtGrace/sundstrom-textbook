Using these definitions, we see that for any set \(A\text{,}\) \(A = A\) and since it is true that for each \(x \in U\text{,}\) if \(x \in A\text{,}\) then \(x \in A\text{,}\) we also see that \(A \subseteq A\text{.}\) That is, any set is equal to itself and any set is a subset of itself. For some specific examples, we see that:

  • \(\displaystyle \left\{ {1, 3, 5} \right\} = \left\{ {3, 5, 1} \right\}\)

  • \(\displaystyle \left\{ {4, 8, 12} \right\} = \left\{ {4, 4, 8, 12, 12} \right\}\)

  • \(\displaystyle \left\{ {5, 10} \right\} = \left\{ {5, 10, 5} \right\}\)

  • \(\displaystyle \left\{ {5, 10} \right\} \ne \left\{ {5, 10, 15} \right\} \text{ but } \left\{ {5, 10} \right\} \subseteq \left\{ {5, 10, 15} \right\} \text{ and } \left\{ {5, 10, 15} \right\} \not \subseteq \left\{ {5, 10} \right\}\)

In each of the first three examples, the two sets have exactly the same elements even though the elements may be repeated or written in a different order.
