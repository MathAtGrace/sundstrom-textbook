Theorem 6.5.1.

Let \(A\) and \(B\) be nonempty sets and let \(f\) be a subset of \(A \times B\) that satisfies the following two properties:

  • For every \(a \in A\text{,}\) there exists \(b \in B\) such that \(( {a, b} ) \in f\text{;}\) and

  • For every \(a \in A\) and every \(b, c \in B\text{,}\) if \(( {a,\;b} ) \in f\) and \(( {a, c} ) \in f\text{,}\) then \(b = c\text{.}\)

If we use \(f( a ) = b\) whenever \(( {a, b} ) \in f\text{,}\) then \(f\) is a function from \(A\) to \(B\text{.}\)
