To prove that \(f\) is a surjection, we let \(y \in ( a, b )\text{.}\) If \(x = \dfrac{y - a}{b - a}\text{,}\) then

\begin{align*} f ( x ) \amp = f \!\left( \frac{y - a}{b - a} \right)\\ \amp = ( b - a ) \!\left( \frac{y - a}{b - a} \right) + a\\ \amp = ( y - a ) + a\\ \amp = y\text{.} \end{align*}
