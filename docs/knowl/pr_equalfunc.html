Progress Check 6.2.4. Equality of Functions.

Let \(A\) be a nonempty set. The identity function on the set \(\boldsymbol{A}\) denoted by \(I_A\) , is the function \(I_A\x A \to A\) defined by \(I_A ( x ) = x\) for every \(x\) in \(A\text{.}\) That is, for the identity map, the output is always equal to the input.

For this progress check, we will use the functions \(f\) and \(g\) from Progress Check 6.2.2. The identity function on the set \(R_5\) is

\begin{equation*} I_{R_5}:R_5 \to R_5 \text{ by } I_{R_5}(x) = x \pmod 5, \text{ for each } x \in R_5\text{.} \end{equation*}

Is the identity function on \(R_5\) equal to either of the functions \(f\) or \(g\) from Progress Check 6.2.2? Explain.
