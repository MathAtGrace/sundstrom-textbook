The other thing to remember is that the elements of a Cartesian product are ordered pairs. So when we start a proof of a result such as Item 2 of Theorem 5.4.6, the primary goal is to prove that the two sets are equal. We will do this by proving that each one is a subset of the other one. So if we want to prove that \(A \times \left( {B \cup C} \right) \subseteq \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{,}\) we can start by choosing an arbitrary element of \(A \times \left( {B \cup C} \right)\text{.}\) The goal is then to show that this element must be in \(\left( {A \times B} \right) \cup \left( {A \times C} \right)\text{.}\) When we start by choosing an arbitrary element of \(A \times \left( {B \cup C} \right)\text{,}\) we could give that element a name. For example, we could start by letting

\begin{equation} u \text{ be an element of } A \times \left( {B \cup C} \right)\!\text{.}\tag{5.11} \end{equation}

We can then use the definition of “ordered pair” to conclude that

\begin{equation} \text{ there exists } x \in A \text{ and there exists } y \in B \cup C \text{ such that } u = \left( {x,y} \right)\text{.}\tag{5.12} \end{equation}

In order to prove that \(A \times \left( {B \cup C} \right) \subseteq \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{,}\) we must now show that the ordered pair \(u\) from (5.11) is in \(\left( {A \times B} \right) \cup \left( {A \times C} \right)\text{.}\) In order to do this, we can use the definition of set union and prove that

\begin{equation} u \in (A \times B) \text{ or } u \in (A \times C)\text{.}\tag{5.13} \end{equation}

Since \(u = (x, y)\text{,}\) we can prove (5.13) by proving that

\begin{equation} \left( x \in A \text{ and } y \in B \right) \text{ or } \left( x \in A \text{ and } y \in C \right)\text{.}\tag{5.14} \end{equation}
