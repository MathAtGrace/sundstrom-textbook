There exists a real number \(x\) such that \(\cos \left( 2x \right) = 2 \left( \cos x \right)\text{.}\) The negation is \(\left( {\forall x \in \mathbb{R}} \right) \left( \cos \left( 2x \right) \ne 2 \left( \cos x \right) \right)\text{,}\) which is, For each real number \(x\text{,}\) \(\cos \left( 2x \right) \ne 2 \left( \cos x \right)\text{.}\)
