Procedure 4.1.4. Procedure for a Proof by Mathematical Induction.

To prove: \(\left( {\forall n \in \mathbb{N}} \right)\left( {P\left( n \right)} \right)\)

Basis step: Prove \(P( 1 )\text{.}\)

Inductive step: Prove that for each \(k \in \mathbb{N}\text{,}\) if \(P( k )\) is true, then \(P( {k + 1} )\) is true.

We can then conclude that \(P( n )\) is true for all \(n \in \mathbb{N}\text{.}\)
