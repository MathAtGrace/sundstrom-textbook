An arrow diagram can be used when the domain and codomain of the function are finite (and small). We represent the elements of each set with points and then use arrows to show how the elements of the domain are associated with elements of the codomain. For example, the arrow from the point 2 in \(A\) to the point \(a\) in \(B\) represents the fact that \(F( 2 ) = a\text{.}\) In this case, we can use the arrow diagram in FigureÂ 6.1.9 to conclude that \(\text{ range} (F) = \{ a, b \}\text{.}\)
