Let \(A = \left\{ {1,2,3} \right\}\text{,}\) \(T = \left\{ {1,2} \right\}\text{,}\) \(B = \left\{ {a,b} \right\}\) , and \(C = \left\{ {a,c} \right\}\text{.}\) We can then form new sets from all of the set operations we have studied. For example, \(B \cap C = \left\{ a \right\}\text{,}\) and so

\begin{equation*} A \times \left( {B \cap C} \right) = \left\{ {\left( {1,a} \right),\left( {2,a} \right),\left( {3,a} \right)} \right\}\text{.} \end{equation*}
