“The function \(\boldsymbol{f}\) is an injection” means that

  • For all \(x_1 , x_2 \in A\text{,}\) if \(x_1 \ne x_2\text{,}\) then \(f( {x_1 } ) \ne f( {x_2 } )\text{;}\) or

  • For all \(x_1 , x_2 \in A\text{,}\) if \(f( {x_1 } ) = f( {x_2 } )\text{,}\) then \(x_1 = x_2\text{.}\)
