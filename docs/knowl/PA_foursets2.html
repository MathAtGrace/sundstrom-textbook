Exercise 2.

Use the roster method to specify each of the following sets. In each case, be sure to follow the order specified by the parentheses.

(a)

\(\left( A \cup B \cup C \right) \cup D\)

(b)

\(A \cup \left( B \cup C \cup D \right)\)

(c)

\(A \cup \left( B \cup C \right) \cup D\)

(d)

\(\left( A \cup B \right) \cup \left( C \cup D \right)\)

(e)

\(\left( A \cap B \cap C \right) \cap D\)

(f)

\(A \cap \left( B \cap C \cap D \right)\)

(g)

\(A \cap \left( B \cap C \right) \cap D\)

(h)

\(\left( A \cap B \right) \cap \left( C \cap D \right)\)
