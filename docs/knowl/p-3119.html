In ExerciseÂ 6, we introduced the number of divisors function \(d\text{.}\) For this function, \(d\x \mathbb{N} \to \mathbb{N}\text{,}\) where \(d( n )\) is the number of natural number divisors of \(n\text{.}\) A function that is related to this function is the so-called set of divisors function. This can be defined as a function \(S\) that associates with each natural number the set of its distinct natural number factors. For example, \(S( 6 ) = \left\{ {1, 2, 3, 6} \right\}\) and \(S(10) = \{1, 2, 5, 10 \}\text{.}\)
