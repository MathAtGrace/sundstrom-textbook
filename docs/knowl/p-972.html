So when we specifically include the universal quantifier, the symbolic form of the negation of a conditional statement is

\begin{align*} \mynot \left( {\forall x} \in U \right)\left[ {P( x ) \to Q( x )} \right] \amp \equiv \left( {\exists x} \in U \right)\mynot \left[ {P( x ) \to Q( x )} \right]\\ \amp \equiv \left( {\exists x} \in U \right)\left[ {P( x ) \wedge \mynot Q( x )} \right] \end{align*}
