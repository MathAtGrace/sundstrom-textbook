When a predicate contains more than one variable, each variable must be quantified to create a statement. For example, assume the universal set is the set of integers, \(\mathbb{Z}\text{,}\) and let \(P\left( {x, y} \right)\) be the predicate, “\(x + y = 0\text{.}\)” We can create a statement from this predicate in several ways.

  1. \(\left( {\forall x \in \mathbb{Z}} \right)\left( {\forall y \in \mathbb{Z}} \right) \left( {x + y = 0} \right)\)

    We could read this as, “For all integers \(x\) and \(y\text{,}\) \(x + y = 0\text{.}\)” This is a false statement since it is possible to find two integers whose sum is not zero \(\left( {2 + 3 \ne 0} \right)\text{.}\)

  2. \(\left( {\forall x \in \mathbb{Z}} \right)\left( {\exists y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\)

    We could read this as, “For every integer \(x\text{,}\) there exists an integer \(y\) such that \(x + y = 0\text{.}\)” This is a true statement.

  3. \(\left( {\exists x \in \mathbb{Z}} \right)\left( {\forall y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\)

    We could read this as, “There exists an integer \(x\) such that for each integer \(y\text{,}\) \(x + y = 0\text{.}\)” This is a false statement since there is no integer whose sum with each integer is zero.

  4. \(\left( {\exists x \in \mathbb{Z}} \right)\left( {\exists y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\)

    We could read this as, “There exist integers \(x\) and \(y\) such that \(x + y = 0\text{.}\)” This is a true statement. For example, \(2 + \left( { - 2} \right) = 0\text{.}\)

When we negate a statement with more than one quantifier, we consider each quantifier in turn and apply the appropriate part of Theorem 2.24. As an example, we will negate Item 3 from the preceding list. The statement is

\begin{equation*} \left( {\exists x \in \mathbb{Z}} \right)\left( {\forall y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\text{.} \end{equation*}

We first treat this as a statement in the following form: \(\left( {\exists x \in \mathbb{Z}} \right)\left( {P( x )} \right)\) where \(P( x )\) is the predicate \(\left( {\forall y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\text{.}\) Using Theorem 2.24, we have

\begin{equation*} \mynot \left( {\exists x \in \mathbb{Z}} \right)\left( {P( x )} \right) \equiv \left( {\forall x \in \mathbb{Z}} \right)\left( {\mynot P( x )} \right)\text{.} \end{equation*}

Using Theorem 2.24 again, we obtain the following:

\begin{align*} \mynot P( x ) \amp \equiv \mynot \left( {\forall y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\\ \amp \equiv \left( {\exists y \in \mathbb{Z}} \right)\mynot \left( {x + y = 0} \right)\\ \amp \equiv \left( {\exists y \in \mathbb{Z}} \right)\left( {x + y \ne 0} \right) \end{align*}

Combining these two results, we obtain

\begin{equation*} \mynot \left( {\exists x \in \mathbb{Z}} \right)\left( {\forall y \in \mathbb{Z}} \right)\left( {x + y = 0} \right) \equiv \left( {\forall x \in \mathbb{Z}} \right)\left( {\exists y \in \mathbb{Z}} \right)\left( {x + y \ne 0} \right)\text{.} \end{equation*}
