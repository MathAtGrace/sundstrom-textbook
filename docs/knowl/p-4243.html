For an example of the use of the notation in Theorem 6.5.7, let \(\R^ + = \left\{ { {x \in \R} \mid x > 0} \right\}\text{.}\) Define

\begin{equation*} f\x \R \to \R \text{ by } f( x ) = x^3; \text{ and } g\x \R \to \R^ + \text{ by }g( x ) = e^x\text{.} \end{equation*}

Notice that \(\R^+\) is the codomain of \(g\text{.}\) We can then say that both \(f\) and \(g\) are bijections. Consequently, the inverses of these functions are also functions.
