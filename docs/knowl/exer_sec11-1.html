Exercise 1.

Which of the following sentences are statements?

(a)

\(3^2 + 4^2 = 5^2\text{.}\)

(b)

\(a^2 + b^2 = c^2\text{.}\)

(c)

There exist integers \(a\text{,}\) \(b\text{,}\) and \(c\) such that \(a^2 = b^2 + c^2\text{.}\)

(d)

If \(x^2 = 4\text{,}\) then \(x = 2\text{.}\)

(e)

For each real number \(x\text{,}\) if \(x^2 = 4\text{,}\) then \(x = 2\text{.}\)

(f)

For each real number \(t\text{,}\) \(\sin ^2 t + \cos ^2 t = 1\text{.}\)

(g)

\(\sin x \lt \sin \! \left( {\dfrac{\pi }{4}} \right)\text{.}\)

(h)

If \(n\) is a prime number, then \(n^2\) has three positive factors.

(i)

\(1 + \tan ^2 \theta = \sec ^2 \theta\text{.}\)

(j)

Every rectangle is a parallelogram.

(k)

Every even natural number greater than or equal to 4 is the sum of two prime numbers.
