For example, since California and Oregon have a land border, we can say that \((\text{ California, Oregon } ) \in R\) and \((\text{ Oregon, California } ) \in R\text{.}\) Also, since California and Michigan do not share a land border, \(\text{ (California, Michigan) } \notin R\) and \((\text{ Michigan, California } ) \notin R\text{.}\)
