Activity 29. Proof of Theorem 5.1.13.

To help with the proof by induction of Theorem 5.1.13, we first prove the following lemma. (The idea for the proof of this lemma was illustrated with the discussion of power set after Definition 5.1.12.)

Lemma 5.1.15.

Let \(A\) and \(B\) be subsets of some universal set. If \(A = B \cup \left\{ x \right\}\text{,}\) where \(x \notin B\text{,}\) then any subset of \(A\) is either a subset of \(B\) or a set of the form \(C \cup \left\{ x \right\}\text{,}\) where \(C\) is a subset of \(B\text{.}\)

Proof.

To begin the induction proof of Theorem 5.1.13, for each nonnegative integer \(n\text{,}\) we let \(P ( n )\) be, “If a finite set has exactly \(n\) elements, then that set has exactly \(2^n\) subsets.”

(a)

Verify that \(P ( 0 )\) is true. (This is the basis step for the induction proof.)

(b)

Verify that \(P( 1 )\) and \(P( 2 )\) are true.

(c)

Now assume that \(k\) is a nonnegative integer and assume that \(P( k )\) is true. That is, assume that if a set has \(k\) elements, then that set has \(2^k\) subsets. (This is the inductive assumption for the induction proof.) Let \(T\) be a subset of the universal set with \(\card (T) = k + 1\text{,}\) and let \(x \in T\text{.}\) Then the set \(B = T - \left\{ x \right\}\) has \(k\) elements. Now use the inductive assumption to determine how many subsets \(B\) has. Then use Lemma 5.1.15 to prove that \(T\) has twice as many subsets as \(B\text{.}\) This should help complete the inductive step for the induction proof.
