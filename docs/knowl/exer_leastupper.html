Activity 10. Least Upper Bound for a Subset of \(\R\).

In Activity 9, we introduced the definition of an upper bound for a subset of the real numbers. Assume that we know this definition and that we know what it means to say that a number is not an upper bound for a subset of the real numbers.

Let \(A\) be a subset of \(\mathbb{R}\text{.}\) A real number \(\alpha\) is the least upper bound for \(A\) provided that \(\alpha\) is an upper bound for \(A\text{,}\) and if \(\beta\) is an upper bound for \(A\text{,}\) then \(\alpha \leq \beta\text{.}\)

Note: The symbol \(\alpha\) is the lowercase Greek letter alpha, and the symbol \(\beta\) is the lowercase Greek letter beta.

If we define \(P\left( x \right)\) to be “\(x\) is an upper bound for \(A\text{,}\)” then we can write the definition for least upper bound as follows:

A real number \(\alpha\) is the least upper bound for \(A\) provided that \(P\left( \alpha \right) \wedge \left[ {\left( {\forall \beta \in \mathbb{R}} \right)\left( {P\left( \beta \right) \to \left( {\alpha \leq \beta } \right)} \right)} \right]\text{.}\)

(a)

Why is a universal quantifier used for the real number \(\beta\text{?}\)

(b)

Complete the following sentence in symbolic form: “A real number \(\alpha\) is not the least upper bound for \(A\) provided that \(\ldots \text{.}\)”

(c)
