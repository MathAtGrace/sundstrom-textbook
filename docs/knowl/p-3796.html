The process of finding the average of a finite set of real numbers can be thought of as a function from \(\mathcal{F} ( \R )\) to \(\R\text{.}\) So the domain is \(\mathcal{F} ( \R )\text{,}\) the codomain is \(\R\text{,}\) and we can define a function \(\text{ avg } : \mathcal{F} ( \R ) \to \R\) as follows: If \(A \in \mathscr{F} ( \R )\) and \(A = \left\{ a_1, a_2, \ldots, a_n \right\}\text{,}\) then \(\text{ avg } ( A ) = \dfrac{a_1 + a_2 + \cdots + a_n}{n}\text{.}\)
