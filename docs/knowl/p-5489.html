Substituting this value for \(y\) in equation (8.15) and solving for \(x\) yields

\begin{equation*} x = x_0 + \frac{b}{d} k\text{.} \end{equation*}

This proves that every solution of the Diophantine equation \(ax + by = c\) can be written in the form prescribed in (8.14).
