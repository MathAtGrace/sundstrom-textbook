Since \(10^3 \equiv 1 \pmod 9\text{,}\) \(10^2 \equiv 1 \pmod 9\) and \(10 \equiv 1 \pmod 9\text{,}\) we can conclude that \([ 10^3 ] = [ 1 ]\text{,}\) \([ {10^2 } ] = [ 1 ]\) and \([ {10} ] = [ 1 ]\text{.}\) Hence, we can use these facts and equation (7.4) to obtain

\begin{align} [ {7319} ] \amp =\left( {[ 7 ] \odot [ {10^3 } ]} \right) \oplus \left( {[ 3 ] \odot [ {10^2} ]} \right) \oplus \left( {[ 1 ] \odot [ 10 ]} \right) \oplus \left( [ 9 ] \odot [ 1 ] \right)\notag\\ \amp = \left( {[ 7 ] \odot [ 1 ]} \right) \oplus \left( {[ 3 ] \odot [ 1 ]} \right) \oplus \left( {[ 1 ] \odot [ 1 ]} \right) \oplus \left( [ 9 ] \odot [ 1 ] \right)\notag\\ \amp = [ 7 ] \oplus [ 3 ] \oplus [ 1 ] \oplus [ 9 ]\notag\\ \amp = [ {7 + 3 + 1 + 9} ]\text{.}\tag{7.5} \end{align}

equation (7.5) tells us that 7319 has the same remainder when divided by 9 as the sum of its digits. It is easy to check that the sum of the digits is 20 and hence has a remainder of 2. This means that when 7319 is divided by 9, the remainder is 2.
