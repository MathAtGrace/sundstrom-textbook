Task 5.b.

Define the relation \(\approx\) on \(\Z_{9}\) as follows: For all \(a, b \in \Z_{9}\text{,}\) \(a \approx b\) if and only if \(a^3 \equiv b^3 \pmod {9}\text{.}\) Prove that \(\approx\) is an equivalence relation on \(\Z_{9}\) and determine all of the distinct equivalence classes of this equivalence relation.
