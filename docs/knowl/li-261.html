Proof

We will use a proof by contradiction. Let us assume that there exist nonzero integers \(a\) and \(b\) such that \(a + 2b = 3\) and \(9a + 2b = 1\) and \(an^3 + 2bn = 3\text{,}\) where \(n\) is a natural number. First, we will solve one equation for \(2b\text{;}\) doing this, we obtain

\begin{align*} a + 2b \amp = 3 \amp\\ 2b \amp = 3 - a \amp (1) \end{align*}

We can now substitute for \(2b\) in \(an^3 + 2bn = 3\text{.}\) This gives

\begin{align*} an^3 + (3 - a)n \amp = 3 \amp\\ an^3 + 3n - an \amp = 3 \amp\\ n \left( an^2 + 3 - a \right) \amp = 3 \amp (2) \end{align*}

By the closure properties of the integers, \(\left( an^2 + 3 - a \right)\) is an integer and, hence, equation (2) implies that \(n\) divides 3. So \(n = 1\) or \(n = 3\text{.}\) When we substitute \(n = 1\) into the equation \(an^3 + 2bn = 3\text{,}\) we obtain \(a + 2b = 3\text{.}\) This is a contradiction since we are told in the proposition that \(a + 2b \ne 3\text{.}\) This proves that the negation of the proposition is false and, hence, the proposition is true.
