Let \(a\) be a positive real number. For all real numbers \(x\) and \(y\text{,}\)

  1. \(\left| x \right| \lt a\) if and only if \(-a \lt x \lt a\text{.}\)

  2. \(\left| xy \right| = \left| x \right| \left| y \right|\text{.}\)

  3. \(\left| x + y \right| \leq \left| x \right| + \left| y \right|\text{.}\) This is known as the Triangle Inequality.
