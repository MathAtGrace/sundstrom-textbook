Answer 5.1.

To prove \(f \left( A \cup B \right) \subseteq f \left( A \right) \cup f \left( B \right)\text{,}\) let \(y \in f \left( A \cup B \right)\text{.}\) Then there exists an \(x \in A \cup B\) such that \(f \left( x \right) = y\text{.}\) Since \(x \in A \cup B\text{,}\) \(x \in A\) or \(x \in B\text{.}\) We first note that if \(x \in A\text{,}\) then \(y = f \left( x \right)\) is in \(f \left( A \right)\text{.}\) In addition, if \(x \in B\text{,}\) then \(y = f \left( x \right)\) is in \(f \left( B \right)\text{.}\) In both cases, \(y = f \left( x \right) \in f \left( A \right) \cup f \left( B \right)\) and hence, \(f \left( A \cup B \right) \subseteq f \left( A \right) \cup f \left( B \right)\text{.}\)

Now let \(y \in f \left( A \right) \cup f \left( B \right)\text{.}\) If \(y \in f \left( A \right)\text{,}\) then there exists an \(x \in A\) such that \(y = f \left( x \right)\text{.}\) Since \(A \subseteq A \cup B\text{,}\) this implies that \(y = f \left( x \right) \in f \left( A \cup B \right)\text{.}\) In a similar manner, we can prove that if \(y \in f \left( B \right)\text{,}\) then \(y \in f \left( A \cup B \right)\text{.}\) Therefore, \(f \left( A \right) \cup f \left( B \right) \subseteq f \left( A \cup B \right)\text{.}\)
