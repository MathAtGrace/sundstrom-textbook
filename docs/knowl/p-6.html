(This is a technical note for instructors.) There has been one change in notation dealing with functions involving congruences starting on Subsection  in Section 6.2 (dated before May 26, 2020). In those copies of Version 2.1, we defined \(\Z_n\) to be the set

\begin{equation*} \Z_n = \{0, 1, 2, \ldots, n-1 \} \end{equation*}

for each natural number \(n\text{.}\) Unfortunately, this could be confused with the definition of the integers modulo \(n\) in Section 7.4, The standard notation for this set of congruence classes is \(\Z_n\text{.}\) So the notation in Section 6.2 has been changed from \(\Z_n\) to \(R_n\text{,}\) which represents the set of all possible remainders upon division by \(n\) in the set of integers. So starting in Section 6.2 and then for the rest of the book, we use the notation

\begin{equation*} R_n = \{0, 1, 2, \ldots, n-1 \}\text{.} \end{equation*}
