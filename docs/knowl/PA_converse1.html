(a)

If \(x = 3\text{,}\) then \(x^2 = 9\text{.}\)

(b)

If \(x^2 = 9\text{,}\) then \(x = 3\text{.}\)

(c)

If \(x^2 \ne 9\text{,}\) then \(x \ne 3\text{.}\)

(d)

If \(x \ne 3\text{,}\) then \(x^2 \ne 9\text{.}\)
