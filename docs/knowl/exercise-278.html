Exercise 13.

Prove the following.

(a)

For each integer \(a\text{,}\) if \(a \not \equiv 0 \pmod 7\text{,}\) then \(a^2 \not \equiv 0 \pmod 7\text{.}\)

(b)

For each integer \(a\text{,}\) if 7 divides \(a^2\text{,}\) then 7 divides \(a\text{.}\)

(c)

The real number \(\sqrt{7}\) is an irrational number.
