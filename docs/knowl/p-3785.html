We must be careful when determining the domain for the derivative function since there are functions that are not differentiable. To make things reasonably easy, we will let \(F\) be the set of all real functions that are differentiable and call this the domain of the derivative function \(D\text{.}\) We will use the set \(T\) of all real functions as the codomain. So our function \(D\) is

\begin{equation*} D\x F \to T \text{ by } D( f ) = f'\text{.} \end{equation*}
