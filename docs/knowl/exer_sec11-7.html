Exercise 7.

Following is a statement of a theorem which can be proven using the quadratic formula. For this theorem, \(a\text{,}\) \(b\text{,}\) and \(c\) are real numbers.

Theorem 1.1.14.

If \(f\) is a quadratic function of the form \(f\left( x \right) = ax^2 + bx + c\) and \(ac \lt 0\text{,}\) then the function \(f\) has two \(x\)-intercepts.

Using only this theorem, what can be concluded about the functions given by the following formulas?

(a)

\(g\left( x \right) = - 8x^2 + 5x - 2\)

(b)

\(h\left( x \right) = - \dfrac{1}{3}x^2 + 3x\)

(c)

\(k\left( x \right) = 8x^2 - 5x - 7\)

(d)

\(j\left( x \right) = - \dfrac{{71}}{{99}}x^2 + 210\)

(e)

\(f\left( x \right) = - 4x^2 - 3x + 7\)

(f)

\(F\left( x \right) = - x^4 + x^3 + 9\)
