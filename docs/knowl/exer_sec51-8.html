Exercise 11.

(a)

Calculate the value of \(5^n - 2^n\) for \(n = 1\text{,}\) \(n = 2\text{,}\) \(n = 3\text{,}\) \(n = 4\text{,}\) \(n = 5\text{,}\) and \(n = 6\text{.}\)

(b)

Based on your work in Task a, make a conjecture about the values of \(5^n - 2^n\) for each natural number \(n\text{.}\)

(c)

Use mathematical induction to prove your conjecture in Task b.
