In SectionÂ 2.1, we used logical operators (conjunction, disjunction, negation) to form new statements from existing statements. In a similar manner, there are several ways to create new sets from sets that have already been defined. In fact, we will form these new sets using the logical operators of conjunction (and), disjunction (or), and negation (not). For example, if the universal set is the set of natural numbers \(\N\) and

\begin{equation*} A = \{ 1, 2, 3, 4, 5, 6 \} \qquad \text{ and } \qquad B = \{ 1, 3, 5, 7, 9 \}\text{,} \end{equation*}

  • The set consisting of all natural numbers that are in \(A\) and are in \(B\) is the set \(\{1, 3, 5 \}\text{;}\)

  • The set consisting of all natural numbers that are in \(A\) or are in \(B\) is the set \(\{ 1, 2, 3, 4, 5, 6, 7, 9 \}\text{;}\) and

  • The set consisting of all natural numbers that are in \(A\) and are not in \(B\) is the set \(\{ 2, 4, 6 \}\text{.}\)
