Corollary 3.47.

If \(n \in \mathbb{N}\text{,}\) then each integer is congruent, modulo \(n\text{,}\) to precisely one of the integers \(0,1,2, \ldots ,n - 1\text{.}\) That is, for each integer \(a\text{,}\) there exists a unique integer \(r\) such that

\begin{equation*} a \equiv r \pmod n \text{ and } 0 \leq r \lt n\text{.} \end{equation*}
