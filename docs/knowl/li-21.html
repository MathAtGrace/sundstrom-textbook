Item. Use of prior knowledge.

This also is very important. We cannot start from square one every time we explore a statement. We must make use of our acquired mathematical knowledge. For the conjecture that \(\sin\left(2x \right) = 2 \sin (x)\text{,}\) for all real numbers \(x\text{,}\) we might recall that there are trigonometric identities called “double angle identities.” We may even remember the correct identity for \(\sin\left( 2x \right)\text{,}\) but if we do not, we can always look it up. We should recall (or find) that

\begin{equation*} \text{ for all real numbers } x, \sin \! \left( 2x \right) = 2 \sin (x) \! \cos (x)\text{.} \end{equation*}

We could use this identity to argue that the conjecture “for all real numbers \(x\text{,}\) \(\sin (2x) = 2 \sin(x)\)” is false, but if we do, it is still a good idea to give a specific counterexample as we did before.
