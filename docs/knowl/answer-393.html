Answer 3.1.

Let \(x \in A\text{.}\) Since \(x\) has the same number of digits as itself, the relation \(\mathrel{R}\) is reflexive. Now let \(x, y, z \in A\text{.}\) If \(x \mathrel{R} y\text{,}\) then \(x\) and \(y\) have the same number of digits. Hence, \(y\) and \(x\) have the same number of digits and \(y \mathrel{R} x\text{,}\) and so \(\mathrel{R}\) is symmetric.

If \(x \mathrel{R} y\) and \(y \mathrel{R} z\text{,}\)then \(x\) and \(y\) have the same number of digits and \(y\) and \(z\) have the same number of digits. Hence, \(x\) and \(z\) have the same number of digits, and so \(x \mathrel{R} z\text{.}\) Therefore, \(\mathrel{R}\) is transitive.

The equivalence classes are: \(\left\{ 0, 1, 2, \ldots , 9 \right\}\text{,}\) \(\left\{ 10, 11, 12, \ldots , 99 \right\}\text{,}\) \(\left\{ 100, 101, 102, \ldots , 999 \right\}\text{,}\) \(\left\{ 1000 \right\}\text{.}\)
