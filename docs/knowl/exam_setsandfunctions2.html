Progress Check 6.45. Set Operations and Functions Acting on Sets.

Define \(f\x \mathbb{R} \to \mathbb{R}\) by \(f ( x ) = x^2 + 2\) for all \(x \in \mathbb{R}\text{.}\) It will be helpful to use the graph shown in Figure 6.46.

We will use the following closed intervals:

\begin{equation*} A = \left[ 0, 3 \right] \qquad B = \left[ -2, 1\right] \qquad C = \left[ 2, 6 \right] \qquad D = \left[ 0, 3 \right] \end{equation*}
(a)

Verify that \(f ( A ) = [2, 11]\text{,}\) \(f ( B ) = [2, 6]\text{,}\) \(f^{-1} ( C ) = [-2, 2]\text{,}\) and that \(f^{-1} ( D ) = [-1, 1]\text{.}\)

(b)

Explain why

(i)

\(f ( A \cap B ) = [2, 3]\) and \(f ( A ) \cap f ( B ) = [2, 6]\text{.}\) So in this case, \(f ( A \cap B ) \subseteq f ( A ) \cap f ( B )\text{.}\)

(ii)

\(f ( A \cup B ) = [2, 11]\) and \(f ( A ) \cup f ( B ) = [2, 11]\text{.}\) So in this case, \(f ( A \cup B ) = f ( A ) \cup f ( B )\text{.}\)

(iii)

\(f^{-1} ( C \cap D ) = [-1, 1]\) and \(f^{-1} ( C ) \cap f^{-1} ( D ) = [-1, 1]\text{.}\) So in this case, \(f^{-1} ( C \cap D ) = f^{-1} ( C ) \cap f^{-1} ( D )\text{.}\)

(iv)

\(f^{-1} ( C \cup D ) = [-2, 2]\) and \(f^{-1} ( C ) \cup f^{-1} ( D ) = [-2, 2]\text{.}\) So in this case, \(f^{-1} ( C \cup D ) = f^{-1} ( C ) \cup f^{-1} ( D )\text{.}\)

(c)

Recall that \(A = [0, 3]\text{.}\) Notice \(f ( A ) = [2, 11]\) is a subset of the codomain, \(\mathbb{R}\text{.}\) Explain why \(f^{-1} \!\left( f ( A ) \right) = [-3, 3]\text{.}\) Since \(f^{-1} \!\left( f ( A ) \right)\) is a subset of the domain, \(\mathbb{R}\text{,}\) we see that in this case, \(A \subseteq f^{-1} \!\left( f ( A ) \right)\text{.}\)

(d)

Recall that \(C = [2, 6]\text{.}\) Notice that \(f^{-1} ( C ) = [-2, 2]\) is a subset of the domain, \(\mathbb{R}\text{.}\) Explain why \(f \!\left( f^{-1} ( C ) \right) = [2, 6]\text{.}\) Since \(f \!\left( f^{-1} ( C ) \right)\) is a subset of the codomain, \(\mathbb{R}\text{,}\) we see that in this case \(f \!\left( f^{-1} ( C ) \right) = C\text{.}\)
