Example 2.33. Negations of Quantified Statements.

Consider the following statement: \(\left( {\forall x \in \mathbb{R}} \right)\left( {x^3 \geq x^2 } \right)\text{.}\)

We can write this statement as an English sentence in several ways. Following are two different ways to do so.

  • For each real number \(x\text{,}\) \(x^3 \geq x^2\text{.}\)

  • If \(x\) is a real number, then \(x^3\) is greater than or equal to \(x^2\text{.}\)

The second statement shows that in a conditional statement, there is often a hidden universal quantifier. This statement is false since there are real numbers \(x\) for which \(x^3\) is not greater than or equal to \(x^2\text{.}\) For example, we could use \(x = - 1\) or \(x = \frac{1}{2}\text{.}\)

This means that the negation must be true. We can form the negation as follows:

\begin{equation*} \mynot \left( {\forall x \in \mathbb{R}} \right)\left( {x^3 \geq x^2 } \right) \equiv \left( {\exists x \in \mathbb{R}} \right)\mynot \left( {x^3 \geq x^2 } \right)\text{.} \end{equation*}

In most cases, we want to write this negation in a way that does not use the negation symbol. In this case, we can now write the open sentence \(\mynot \left( {x^3 \geq x^2 } \right)\) as \(\left( {x^3 \lt x^2 } \right)\text{.}\) (That is, the negation of “is greater than or equal to” is “is less than.”) So we obtain the following:

\begin{equation*} \mynot \left( {\forall x \in \mathbb{R}} \right)\left( {x^3 \geq x^2 } \right) \equiv \left( {\exists x \in \mathbb{R}} \right)\left( {x^3 \lt x^2 } \right)\text{.} \end{equation*}

The statement \(\left( {\exists x \in \mathbb{R}} \right)\left( {x^3 \lt x^2 } \right)\) could be written in English as follows:

There exists a real number \(x\) such that \(x^3 \lt x^2\text{.}\)
There exists an \(x\) such that \(x\) is a real number and \(x^3 \lt x^2\text{.}\)
