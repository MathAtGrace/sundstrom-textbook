We will now consider the following situation: Let \(S\) and \(T\) be sets and let \(f\) be a function from \(S\) to \(T\text{.}\) Also, let \(A\) and \(B\) be subsets of \(S\) and let \(C\) and \(D\) be subsets of \(T\text{.}\) In the remainder of this section, we will consider the following situations and answer the questions posed in each case.

  • The set \(A \cap B\) is a subset of \(S\) and so \(f ( A \cap B )\) is a subset of \(T\text{.}\) In addition, \(f ( A )\) and \(f ( B )\) are subsets of \(T\text{.}\) Hence, \(f ( A ) \cap f ( B )\) is a subset of \(T\text{.}\)

    Is there any relationship between \(f ( A \cap B )\) and \(f ( A ) \cap f ( B )\text{?}\)

  • The set \(A \cup B\) is a subset of \(S\) and so \(f ( A \cup B )\) is a subset of \(T\text{.}\) In addition, \(f ( A )\) and \(f ( B )\) are subsets of \(T\text{.}\) Hence, \(f ( A ) \cup f ( B )\) is a subset of \(T\text{.}\)

    Is there any relationship between \(f ( A \cup B )\) and \(f ( A ) \cup f ( B )\text{?}\)

  • The set \(C \cap D\) is a subset of \(T\) and so \(f^{-1} ( C \cap D )\) is a subset of \(S\text{.}\) In addition, \(f^{-1} ( C )\) and \(f^{-1} ( D )\) are subsets of \(S\text{.}\) Hence, \(f^{-1} ( C ) \cap f^{-1} ( D )\) is a subset of \(S\text{.}\)

    Is there any relationship between the sets \(f^{-1} ( C \cap D )\) and \(f^{-1} ( C ) \cap f^{-1} ( D )\text{?}\)

  • The set \(C \cup D\) is a subset of \(T\) and so \(f^{-1} ( C \cup D )\) is a subset of \(S\text{.}\) In addition, \(f^{-1} ( C )\) and \(f^{-1} ( D )\) are subsets of \(S\text{.}\) Hence, \(f^{-1} ( C ) \cup f^{-1} ( D )\) is a subset of \(S\text{.}\)

    Is there any relationship between the sets \(f^{-1} ( C \cup D )\) and \(f^{-1} ( C ) \cup f^{-1} ( D )\text{?}\)
