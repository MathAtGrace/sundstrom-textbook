First, assume that \(x \in [ a ]\text{.}\) Then, by definition, \(x \sim a\text{.}\) Since we have assumed that \(a \sim b\text{,}\) we can use the transitive property of \(\sim\) to conclude that \(x \sim b\text{,}\) and this means that \(x \in [ b ]\text{.}\) This proves that \([ a ] \subseteq [ b ]\text{.}\)
