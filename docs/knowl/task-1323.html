Task 36.b.

For each of the following functions, determine if the function is an injection and determine if the function is a surjection. Justify all conclusions.

(i)

\(g\x[0, 1] \to (0, 1)\) by

\begin{equation} g(x) = \begin{cases}0.8, \amp \text{ if \(x = 0\); } \\ 0.5 x, \amp \text{ if \(0 \lt x \lt 1\); } \\ 0.6 \amp \text{ if \(x = 1\). } \end{cases}\tag{6.3} \end{equation}
(ii)

\(h\x\Z \to \left\{ 0, 1 \right\}\) by

\begin{equation} h(x) = \begin{cases}0, \amp \text{ if \(x\) is even; } \\ 1, \amp \text{ if \(x\) is odd. } \end{cases}\tag{6.4} \end{equation}
