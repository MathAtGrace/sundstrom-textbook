Let \(x \in A\text{.}\) Since \(x\) has the same number of digits as itself, the relation \(\mathrel{R}\) is reflexive. Now let \(x, y, z \in A\text{.}\) If \(x \mathrel{R} y\text{,}\) then \(x\) and \(y\) have the same number of digits. Hence, \(y\) and \(x\) have the same number of digits and \(y \mathrel{R} x\text{,}\) and so \(\mathrel{R}\) is symmetric.
