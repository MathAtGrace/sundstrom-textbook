Exercise 2.

Identify the hypothesis and the conclusion for each of the following conditional statements.

(a)

If \(n\) is a prime number, then \(n^2\) has three positive factors.

(b)

If \(a\) is an irrational number and \(b\) is an irrational number, then \(a \cdot b\) is an irrational number.

(c)

If \(p\) is a prime number, then \(p = 2\) or \(p\) is an odd number.

(d)

If \(p\) is a prime number and \(p \ne 2\text{,}\) then \(p\) is an odd number.

(e)

If \(p \ne 2\) and \(p\) is an even number, then \(p\) is not prime.
