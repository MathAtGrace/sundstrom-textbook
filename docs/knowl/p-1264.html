As was noted in Section 2.2, there are several different logical equivalencies. Fortunately, there are only a small number that we often use when trying to write proofs, and many of these are listed in Theorem 2.2.7 at the end of Section 2.2. We will illustrate the use of one of these logical equivalencies with the following proposition:

For all real numbers \(a\) and \(b\text{,}\) if \(a \ne 0\) and \(b \ne 0\text{,}\) then \(ab \ne 0\text{.}\)
