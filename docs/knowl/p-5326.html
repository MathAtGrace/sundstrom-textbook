Let \(a\) and \(b\) be nonzero integers, and let \(p\) be a prime number.

  1. If \(a\) and \(b\) are relatively prime, then there exist integers \(m\) and \(n\) such that \(am + bn = 1\text{.}\) That is, 1 can be written as linear combination of \(a\) and \(b\text{.}\)

  2. If \(p \mid a\text{,}\) then \(\gcd( {a, p} ) = p\text{.}\)

  3. If \(p\) does not divide \(a\text{,}\) then \(\gcd( {a, p} ) = 1\text{.}\)
