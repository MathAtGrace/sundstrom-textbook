Activity 21. The Importance of the Basis Step.

Most of the work done in constructing a proof by induction is usually in proving the inductive step. This was certainly the case in Proposition 4.5. However, the basis step is an essential part of the proof. Without it, the proof is incomplete. To see this, let \(P( n )\) be

\begin{equation*} 1 + 2 + \cdots + n = \frac{{n^2 + n + 1}}{2}\text{.} \end{equation*}
(a)

Let \(k \in \mathbb{N}\text{.}\) Complete the following proof that if \(P( k )\) is true, then \(P( k + 1 )\) is true. Let \(k \in \mathbb{N}\text{.}\) Assume that \(P( k )\) is true. That is, assume that

\begin{equation} 1 + 2 + \cdots + k = \frac{{k^2 + k + 1}}{2}\text{.}\tag{4.11} \end{equation}

The goal is to prove that \(P( k + 1 )\) is true. That is, we need to prove that

\begin{equation} 1 + 2 + \cdots + k + \left( {k + 1} \right) = \frac{{\left( {k + 1} \right)^2 + \left( {k + 1} \right) + 1}}{2}\text{.}\tag{4.12} \end{equation}

To do this, we add \(\left( {k + 1} \right)\) to both sides of equation (4.11). This gives

\begin{align*} 1 + 2 + \cdots + k + \left( {k + 1} \right) \amp = \frac{{k^2 + k + 1}}{2} + \left( {k + 1} \right)\\ \amp = \cdots\text{.} \end{align*}
(b)

Is \(P( 1 )\) true? Is \(P( 2 )\) true? What about \(P( 3 )\) and \(P( 4 )\text{?}\) Explain how this shows that the basis step is an essential part of a proof by induction.
