Theorem 5.23.

Let \(A\text{,}\) \(B\text{,}\) and \(C\) be subsets of some universal set \(U\text{.}\) Then

  • \(A \cap B \subseteq A\text{ and } A \subseteq A \cup B\text{.}\)

  • If \(A \subseteq B\text{,}\) then \(A \cap C \subseteq B \cap C\) and \(A \cup C \subseteq B \cup C\text{.}\)

Proof.

The first part of this theorem was included in Exercise 7 from Section 5.2. The second part of the theorem was Exercise 12 from Section 5.2.
