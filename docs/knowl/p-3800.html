In this case, the \(n^\text{ th }\) term of the sequence is \(\dfrac{1}{n}\text{.}\) If we know a formula for the \(n\)th term, we often use this formula to represent the sequence. For example, we might say Define the sequence \(\left\langle {a_n } \right\rangle\) by \(a_n = \dfrac{1}{n}\) for each \(n \in \mathbb{N}\text{.}\) This shows that this sequence is a function with domain \(\mathbb{N}\text{.}\) If it is understood that the domain is \(\N\text{,}\) we could refer to this as the sequence \(\left\langle \dfrac{1}{n} \right \rangle\text{.}\) Given an element of the domain, we can consider \(a_n\) to be the output. In this case, we have used subscript notation to indicate the output rather than the usual function notation. We could just as easily write

\begin{equation*} a( n ) = \frac{1}{n} \text{ instead of } a_n = \frac{1}{n}\text{.} \end{equation*}
