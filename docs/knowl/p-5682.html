If we define the function \(f\) by \(f ( 1 ) = a\text{,}\) \(f ( 2 ) = b\text{,}\) \(f ( 3 ) = c\text{,}\) \(f ( 4 ) = a\text{,}\) and \(f ( 5 ) = b\text{,}\) then we can use \(g ( a ) = 1\text{,}\) \(g ( b ) = 2\text{,}\) and \(g ( 3 ) = c\text{.}\) The function \(g\) is an injection.
