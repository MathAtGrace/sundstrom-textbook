Exercise 10.

For \(\left(a, b \right), \left( c, d \right) \in \R \times \R\text{,}\) define \(\left(a, b \right) \sim \left(c, d \right)\) if and only if \(a^2 + b^2 = c^2 + d^2\text{.}\) In Exercise 15 of Section 7.2, we proved that \(\sim\) is an equivalence relation on \(\R \times \R\text{.}\)

(a)

Determine the equivalence class of \(\left(0, 0 \right)\text{.}\)

(b)

Use set builder notation (and do not use the symbol \(\sim\)) to describe the equivalence class of \(\left(2, 3 \right)\) and then give a geometric description of this equivalence class.

(c)

Give a geometric description of a typical equivalence class for this equivalence relation.

(d)

Let \(\R^* = \left\{ x \in \R \mid x \geq 0 \right\}\text{.}\) Prove that there is a one-to-one correspondence (bijection) between \(\R^*\) and the set of all equivalence classes for this equivalence relation.
