Activity 9. Upper Bounds for Subsets of \(\mathbb{R}\).

Let \(A\) be a subset of the real numbers. A number \(b\) is called an upper bound for the set \(A\) provided that for each element \(x\) in \(A\text{,}\) \(x \leq b\text{.}\)

(a)

Write this definition in symbolic form by completing the following: Let \(A\) be a subset of the real numbers. A number \(b\) is called an upper bound for the set \(A\) provided that \(\ldots \text{.}\)

(b)

Give examples of three different upper bounds for the set \(A = \left\{ x \in \mathbb{R} \mid 1 \leq x \leq 3 \right\}\text{.}\)

(c)

Does the set \(B = \left\{ x \in \mathbb{R} \mid x > 0 \right\}\) have an upper bound? Explain.

(d)

Give examples of three different real numbers that are not upper bounds for the set \(A = \left\{ x \in \mathbb{R} \mid 1 \leq x \leq 3 \right\}\text{.}\)

(e)

Complete the following in symbolic form: “Let \(A\) be a subset of \(\R\text{.}\) A number \(b\) is not an upper bound for the set \(A\) provided that \(\ldots \text{.}\)”

(f)

Without using the symbols for quantifiers, complete the following sentence: “Let \(A\) be a subset of \(\R\text{.}\) A number \(b\) is not an upper bound for the set \(A\) provided that \(\ldots \text{.}\)”

(g)

Are your examples in Task 9.d consistent with your work in Task 9.f? Explain.
