What do you need to do in order to prove that \(n^3\) is odd? Notice that if \(n\) is an odd integer, then there exists an integer \(k\) such that \(n = 2k + 1\text{.}\) Remember that to prove that \(n^3\) is an odd integer, you need to prove that there exists an integer \(q\) such that \(n^3 = 2q + 1\text{.}\) This can also be approached as follows: If \(n\) is odd, then by TheoremÂ 1.13, \(n^2\) is odd.
