Conjecture.

For each integer \(n\text{,}\) if 5 divides \(\left(n^2 - 1 \right)\text{,}\) then 5 divides \(\left( n - 1 \right)\text{.}\)

The integer \(n = 4\) is a counterexample that proves this conjecture is false. Notice that when \(n = 4\text{,}\) \(n^2 - 1 = 15\) and 5 divides 15. Hence, the hypothesis of the conjecture is true in this case. In addition, \(n - 1 = 3\) and 5 does not divide 3 and so the conclusion of the conjecture is false in this case. Since this is an example where the hypothesis is true and the conclusion is false, the conjecture is false.
