Exercise 7.

Assume that the universal set is \(\mathbb{R}\text{.}\) Consider the following sentence:

\begin{equation*} \left( {\exists t \in \mathbb{R}} \right)\left( {t \cdot x = 20} \right)\text{.} \end{equation*}
(a)

Explain why this sentence is an open sentence and not a statement.

(b)

If 5 is substituted for \(x\text{,}\) is the resulting sentence a statement? If it is a statement, is the statement true or false?

(c)

If \(\pi\) is substituted for \(x\text{,}\) is the resulting sentence a statement? If it is a statement, is the statement true or false?

(d)

If \(0\) is substituted for \(x\text{,}\) is the resulting sentence a statement? If it is a statement, is the statement true or false?

(e)

What is the truth set of the open sentence \(\left( {\exists t \in \mathbb{R}} \right)\left( {t \cdot x = 20} \right)\text{?}\)
