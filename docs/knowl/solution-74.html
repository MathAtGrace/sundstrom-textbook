Solution 3.1.12.e.1.

Let \(a\) and \(b\) be integers and assume that \(\mod{a}{5}{8}\) and \(\mod{b}{5}{8}\text{.}\) We will prove that \(\mod{(a + b)}{2}{8}\text{.}\) Since 8 divides \((a - 5)\) and \((b - 5)\text{,}\) there exist integers \(k\) and \(m\) such that \(a - 5 = 8k\) and \(b - 5 = 8m\text{.}\) We then see that

\begin{align*} a + b - 2 \amp = (5 + 8k) + (5 + 8m) - 2\\ \amp = 8 + 8k + 8m\\ \amp = 8(1 + k + m) \end{align*}

By the closure properties of the integers, \((1 + k + m)\) is an integer and so the last equation proves that 8 divides \((a + b - 2)\) and hence, \(\mod{(a + b)}{2}{8}\text{.}\) This proves that if \(\mod{a}{5}{8}\) and \(\mod{b}{5}{8}\text{,}\) then \(\mod{(a + b)}{2}{8}\text{.}\)
