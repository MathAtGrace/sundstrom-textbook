Item 4.

Answering a backward question can sometimes be tricky. If the goal is the statement \(Q\text{,}\) we must construct the know-show table so that if we know that \(Q1\) is true, then we can conclude that \(Q\) is true. It is sometimes easy to answer this in a way that if it is known that \(Q\) is true, then we can conclude that \(Q1\) is true. For example, suppose the goal is to prove

\begin{equation*} y^2 = 4\text{,} \end{equation*}

where \(y\) is a real number. A backward question could be, “How do we prove the square of a real number equals four?” One possible answer is to prove that the real number equals 2. Another way is to prove that the real number equals \(-2\text{.}\) This is an appropriate backward question, and these are appropriate answers.

However, if the goal is to prove

\begin{equation*} y = 2\text{,} \end{equation*}

where \(y\) is a real number, we could ask, “How do we prove a real number equals 2?” It is not appropriate to answer this question with “prove that the square of the real number equals 4.” This is because if \(y^2=4\text{,}\) then it is not necessarily true that \(y=2\text{.}\)
