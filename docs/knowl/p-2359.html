Proposition

For each natural number \(n\text{,}\) \(1 + 4 + 7 + \cdots + \left( 3n - 2 \right) = \dfrac{n \left(3n - 1 \right)}{2}\text{.}\)

Proof

We will prove this proposition using mathematical induction. So we let \(P ( n )\) be the open sentence

\begin{equation*} 1 + 4 + 7 + \cdots + \left( 3n - 2 \right)\!\text{.} \end{equation*}

Using \(n = 1\text{,}\) we see that \(3n - 2 = 1\) and hence, \(P \left( 1 \right)\) is true.

We now assume that \(P( k )\) is true. That is,

\begin{equation*} 1 + 4 + 7 + \cdots + \left( 3k - 2 \right) = \frac{k \left(3k -1 \right)}{2}\text{.} \end{equation*}

We then see that

\begin{align*} 1 + 4 + 7 + \cdots + \left( 3k - 2 \right) + \left( 3 (k + 1) - 2 \right) \amp = \frac{\left(k + 1 \right) \left(3k + 2 \right)}{2}\\ \frac{k \left(3k -1 \right)}{2} + \left(3k + 1 \right) \amp = \frac{\left(k + 1 \right) \left(3k + 2 \right)}{2}\\ \frac{\left(3k^2 - k \right) + \left(6k + 2 \right) }{2} \amp = \frac{3k^2 + 5k + 2}{2}\\ \frac{3k^2 +5k + 2}{2} \amp = \frac{3k^2 + 5k + 2}{2}\text{.} \end{align*}

We have thus proved that \(P( k + 1 )\) is true, and hence, we have proved the proposition.
