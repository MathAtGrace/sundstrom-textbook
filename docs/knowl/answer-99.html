Answer 3.e.1.

For each integer \(x\text{,}\) if \(x^2\) is odd, then \(x\) is odd. The negation is \(\left( \exists x \in \mathbb{Z} \right) \left( x^2 \text{ is odd and \(x\) is even } \right)\text{,}\) which is, There exists an integer \(x\) such that \(x^2\) is odd and \(x\) is even.
