We assume that \(m\) is an odd integer and will prove that \(\left( 3m^2 + 4m + 6 \right)\) is an odd integer. Since \(m\) is an odd integer, there exists an integer \(k\) such that \(m = 2k + 1\text{.}\) Substituting this into the expression \(\left( 3m^2 + 4m + 6 \right)\) and using algebra, we obtain

\begin{align*} 3m^2 + 4m + 6 \amp = 3\left(2k + 1 \right)^2 + 4 \left(2k + 1 \right) + 6\\ \amp = \left( 12k^2 + 12k + 3 \right) + \left( 8k + 4 \right) + 6\\ \amp = 12k^2 + 20k + 13\\ \amp = 12k^2 + 20k + 12 + 1\\ \amp = 2 \left(6k^2 + 10k + 6 \right) + 1 \end{align*}
