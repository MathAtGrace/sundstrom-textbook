Note: A Caution about Notation. The standard notation for an open interval in \(\mathbb{R}\) is the same as the notation for an ordered pair, which is an element of \mbox{\(\mathbb{R} \times \mathbb{R}\)} . We need to use the context in which the notation is used to determine which interpretation is intended. For example,

  • If we write \(\left( {\sqrt 2 ,7} \right) \in \mathbb{R} \times \mathbb{R}\text{,}\) then we are using \(\left( {\sqrt 2 ,7} \right)\) to represent an ordered pair of real numbers.

  • If we write \(\left( {1,2} \right) \times \left\{ 4 \right\}\text{,}\) then we are interpreting \(\left( {1,2} \right)\) as an open interval. We could write

    \begin{equation*} \left( {1,2} \right) \times \left\{ 4 \right\} = \left\{ {\left. {\left( {x,4} \right)\,} \right|1 \lt x \lt 2} \right\}\!\text{.} \end{equation*}
