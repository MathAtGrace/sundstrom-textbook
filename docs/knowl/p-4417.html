We assume that \(y \in f ( A \cap B )\text{.}\) This means that there exists an \(x \in A \cap B\) such that \(f ( x ) = y\text{.}\) Since \(x \in A \cap B\text{,}\) we conclude that \(x \in A\) and \(x \in B\text{.}\)

  • Since \(x \in A\) and \(f ( x ) = y\text{,}\) we conclude that \(y \in f ( A )\text{.}\)

  • Since \(x \in B\) and \(f ( x ) = y\text{,}\) we conclude that \(y \in f ( B )\text{.}\)
