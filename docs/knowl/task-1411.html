Task 6.6.5.c.

For each of the following, determine the two subsets of \(R_8\) and then determine if there is a relationship between the two sets. For example, \(A \cap B = \left\{ 4 \right\}\) and since \(f(4) = 2\text{,}\) we see that \(f ( A \cap B ) = \left\{ 2 \right\}\text{.}\)

(i)

\(f ( A \cap B )\) and \(f ( A ) \cap f ( B )\)

(ii)

\(f ( A \cup B )\) and \(f ( A ) \cup f ( B )\)

(iii)

\(f^{-1} ( C \cap D )\) and \(f^{-1} ( C ) \cap f^{-1} ( D )\)

(iv)

\(f^{-1} ( C \cup D )\) and \(f^{-1} ( C ) \cup f^{-1} ( D )\)
