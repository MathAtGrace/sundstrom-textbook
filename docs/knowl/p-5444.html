We know that any real number \(a\) can be represented in decimal form as follows:

\begin{equation*} a = A.a_1 a_2 a_3 a_4 \cdots a_n \cdots\text{,} \end{equation*}

where \(A\) is an integer and the decimal part \(( 0.a_1 a_2 a_3 a_4 \cdots )\) is in normalized form. (See [cross-reference to target(s) "normalizedform" missing or not unique].) We also know that the real number \(a\) is an irrational number if and only \(a\) has an infinite non-repeating decimal expansion. We now associate with \(a\) the real number

\begin{equation} A.a_1 0 a_2 1 1 a_3 0 0 0 a_4 1 1 1 1 a_5 0 0 0 0 0 a_6 1 1 1 1 1 1 \cdots\text{.}\tag{9.1} \end{equation}

Notice that to construct the real number in equationÂ (9.1), we started with the decimal expansion of \(a\text{,}\) inserted a 0 to the right of the first digit after the decimal point, inserted two 1's to the right of the second digit to the right of the decimal point, inserted three 0's to the right of the third digit to the right of the decimal point, and so on.
