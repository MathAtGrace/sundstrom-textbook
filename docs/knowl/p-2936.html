To help with the proof by induction of Theorem 5.9, we first prove the following lemma. (The idea for the proof of this lemma was illustrated with the discussion of power set after Assemblage .)

Lemma 5.10.

Let \(A\) and \(B\) be subsets of some universal set. If \(A = B \cup \left\{ x \right\}\text{,}\) where \(x \notin B\text{,}\) then any subset of \(A\) is either a subset of \(B\) or a set of the form \(C \cup \left\{ x \right\}\text{,}\) where \(C\) is a subset of \(B\text{.}\)

Proof.

Let \(A\) and \(B\) be subsets of some universal set, and assume that \(A = B \cup \left\{ x \right\}\) where \(x \notin B\text{.}\) Let \(Y\) be a subset of \(A\text{.}\) We need to show that \(Y\) is a subset of \(B\) or that \(Y = C \cup \left\{ x \right\}\text{,}\) where \(C\) is some subset of \(B\text{.}\) There are two cases to consider: (1) \(x\) is not an element of \(Y\text{,}\) and (2) \(x\) is an element of \(Y\text{.}\)

Case 1: Assume that \(x \notin Y\text{.}\) Let \(y \in Y\text{.}\) Then \(y \in A\) and \(y \ne x\text{.}\) Since

\begin{equation*} A = B \cup \left\{ x \right\}\!\text{,} \end{equation*}

this means that \(y\) must be in \(B\text{.}\) Therefore, \(Y \subseteq B\text{.}\)

Case 2: Assume that \(x \in Y\text{.}\) In this case, let \(C = Y - \left\{ x \right\}\text{.}\) Then every element of \(C\) is an element of \(B\text{.}\) Hence, we can conclude that \(C \subseteq B\) and that \(Y = C \cup \left\{ x \right\}\text{.}\)

Cases (1) and (2) show that if \(Y \subseteq A\text{,}\) then \(Y \subseteq B\) or \(Y = C \cup \left\{ x \right\}\text{,}\) where \(C \subseteq B\text{.}\)
