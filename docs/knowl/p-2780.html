The integers consist of the natural numbers, the negatives of the natural numbers, and zero. If we let \(\N^- = \left\{ \ldots, -4, -3, -2, -1 \right\}\text{,}\) then we can use set union and write

\begin{equation*} \Z = \N^- \cup \left\{ 0 \right\} \cup \N\text{.} \end{equation*}

So we see that \(\N \subseteq \Z\text{,}\) and in fact, \(\N \subset \Z\text{.}\)
