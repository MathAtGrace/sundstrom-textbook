Exercise 11.

Let \(f\x S \to T\) be a function, let \(A \subseteq S\text{,}\) and let \(C \subseteq T\text{.}\)

(a)

Item 1 of Theorem 6.50 states that \(A \subseteq f^{-1} \!\left( f ( A ) \right)\text{.}\) Give an example where \(f^{-1} \!\left( f ( A ) \right) \not \subseteq A\text{.}\)

(b)

Item 2 of Theorem 6.50 states that \(f \!\left( f^{-1} ( C ) \right) \subseteq C\text{.}\) Give an example where \(C \not \subseteq f \!\left( f^{-1} ( C ) \right)\text{.}\)
