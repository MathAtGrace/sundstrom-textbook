Exercise 2.

Let \(R_6 = \left\{ {0, 1, 2, 3, 4, 5} \right\}\text{.}\) Define \(f\x R_6 \to R_6\) by \(f( x ) = x^2 + 4 \pmod 6\text{,}\) and define \(g\x R_6 \to R_6\) by \(g( x ) = (x + 1)(x + 4) \pmod 6\text{.}\)

(a)

Calculate \(f( 0 )\text{,}\) \(f( 1 ), f( 2 )\text{,}\) \(f( 3 )\text{,}\) \(f( 4 )\text{,}\) and \(f( 5 )\text{.}\)

(b)

Calculate \(g( 0 )\text{,}\) \(g( 1 ), g( 2 )\text{,}\) \(g( 3 )\text{,}\) \(g( 4 )\text{,}\) and \(g( 5 )\text{.}\)

(c)

Is the function \(f\) equal to the function \(g\text{?}\) Explain.
