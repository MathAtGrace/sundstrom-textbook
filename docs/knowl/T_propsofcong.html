Theorem 3.5.5. Properties of Congruence Modulo.

Let \(n\) be a natural number and let \(a, b, c, \text{ and } d\) be integers. If \(a \equiv b \pmod n\) and \(c \equiv d \pmod n\text{,}\) then

  1. \(\left( {a + c} \right) \equiv \left( {b + d} \right) \pmod n\text{.}\)

  2. \(ac \equiv bd \pmod n\text{.}\)

  3. For each \(m \in \mathbb{N}\text{,}\) \(a^m \equiv b^m \pmod n\text{.}\)
