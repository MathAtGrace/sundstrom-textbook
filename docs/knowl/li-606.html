Item 1.

We use the notation \([ a ]\) when only one equivalence relation is being used. If there is more than one equivalence relation, then we need to distinguish between the equivalence classes for each relation. We often use something like \([ a ]_{ \sim}\text{,}\) or if \(R\) is the name of the relation, we can use \(R[ a ]\) or \([a]_R\) for the equivalence class of \(a\) determined by \(R\text{.}\) In any case, always remember that when we are working with any equivalence relation on a set \(A\) if \(a \in A\text{,}\) then the equivalence class \([ a ]\) is a subset of \(A\text{.}\)
