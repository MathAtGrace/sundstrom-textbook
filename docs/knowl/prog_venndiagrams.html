Progress Check 5.1.11. Using Venn Diagrams.

Let \(A\text{,}\) \(B\text{,}\) and \(C\) be subsets of a universal set \(U\text{.}\)

(a)
(i)

\((A \cap B) \cap C\)

(ii)

\((A \cap B) \cup C\)

(iii)

\(\left( A^c \cup B \right)\)

(iv)

\(A^c \cap (B \cup C)\)

(b)

Draw the most general Venn diagram showing \(B \subseteq (A \cup C)\text{.}\)

(c)

Draw the most general Venn diagram showing \(A \subseteq \left( B^c \cup C \right)\text{.}\)
