The primary use of the Principle of Mathematical Induction is to prove statements of the form

\begin{equation*} \left( {\forall n \in \mathbb{N}} \right)\left( {P\left( n \right)} \right)\text{,} \end{equation*}

where \(P( n )\) is some open sentence. Recall that a universally quantified statement like the preceding one is true if and only if the truth set \(T\) of the open sentence \(P( n )\) is the set \(\N\text{.}\) So our goal is to prove that \(T = \N\text{,}\) which is the conclusion of Principle 4.1.3 To verify the hypothesis of the Principle 4.1.3, we must

  1. Prove that \(1 \in T\text{.}\) That is, prove that \(P( 1 )\) is true.

  2. Prove that if \(k \in T\text{,}\) then \(\left( {k + 1} \right) \in T\text{.}\) That is, prove that if \(P( k )\) is true, then \(P( {k + 1} )\) is true.

The first step is called the basis step or the initial step, and the second step is called the inductive step. This means that a proof by mathematical induction will have the following form:

Example 4.1.4. Procedure for a Proof by Mathematical Induction.

To prove: \(\left( {\forall n \in \mathbb{N}} \right)\left( {P\left( n \right)} \right)\)

Basis step:

Prove \(P( 1 )\text{.}\)

Inductive step:

Prove that for each \(k \in \mathbb{N}\text{,}\) if \(P( k )\) is true, then \(P( {k + 1} )\) is true.

We can then conclude that \(P( n )\) is true for all \(n \in \mathbb{N}\text{.}\)
