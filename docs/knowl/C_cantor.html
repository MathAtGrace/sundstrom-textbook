Corollary 9.39.

\(\mathcal{P} ( \mathbb{N} )\) is an infinite set that is not countably infinite.

Proof.

Since \(\mathcal{P} ( \mathbb{N} )\) contains the infinite subset \(\left\{ \left\{ 1 \right\}, \left\{ 2 \right\}, \left\{ 3 \right\} \ldots \,\right\}\text{,}\) we can use Theorem 9.13, to conclude that \(\mathcal{P} ( \mathbb{N} )\) is an infinite set. By Cantor's Theorem (Theorem 9.38), \(\mathbb{N}\) and \(\mathcal{P} ( \mathbb{N} )\) do not have the same cardinality. Therefore, \(\mathcal{P} ( \mathbb{N} )\) is not countable and hence is an uncountable set. \(\blacksquare\)
