Exercise 8.

Let \(\Lambda\) be a nonempty indexing set and let \(\mathscr{A} = \left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) be an indexed family of sets.

(a)

Prove that if \(B\) is a set such that \(B \subseteq A_\alpha\) for every \(\alpha \in \Lambda\text{,}\) then \(B \subseteq \bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha\text{.}\)

(b)

Prove that if \(C\) is a set such that \(A_\alpha \subseteq C\) for every \(\alpha \in \Lambda\text{,}\) then \(\bigcup\limits_{\alpha \in \Lambda}^{}A_\alpha \subseteq C\text{.}\)
