Section 5.5 Indexed Families of Sets

Beginning Activity Beginning Activity 1: The Union and Intersection of a Family of Sets

In Section 5.3, we discussed various properties of set operations. We will now focus on the associative properties for set union and set intersection. Notice that the definition of “set union” tells us how to form the union of two sets. It is the associative law that allows us to discuss the union of three sets. Using the associate law, if \(A\text{,}\) \(B\text{,}\) and \(C\) are subsets of some universal set, then we can define \(A \cup B \cup C\) to be \(\left( A \cup B \right) \cup C\) or \(A \cup \left( B \cup C \right)\text{.}\) That is,

\begin{equation*} A \cup B \cup C = \left(A \cup B \right) \cup C = A \cup \left( B \cup C \right)\!\text{.} \end{equation*}

For this activity, the universal set is \(\N\) and we will use the following four sets:

\begin{align*} A \amp = \left\{ 1, 2, 3, 4, 5 \right\} \amp B \amp = \left\{ 2, 3, 4, 5, 6 \right\}\\ C \amp = \left\{ 3, 4, 5, 6, 7 \right\} \amp D \amp = \left\{ 4, 5, 6, 7, 8 \right\} \end{align*}

We have already seen that the elements of a set may themselves be sets. For example, the power set of a set \(T\text{,}\) \(\mathcal{P}( T )\text{,}\) is the set of all subsets of \(T\text{.}\) The phrase, “a set of sets” sounds confusing, and so we often use the terms collection and family when we wish to emphasize that the elements of a given set are themselves sets. We would then say that the power set of \(T\) is the family (or collection) of sets that are subsets of \(T\text{.}\)

One of the purposes of the work we have done so far in this activity was to show that it is possible to define the union and intersection of a family of sets.

Definition.

Let \(\mathcal{C}\) be a family of sets. The union of \(\mathbf{\mathcal{C}}\) is defined as the set of all elements that are in at least one of the sets in \(\mathcal{C}\text{.}\) We write

\begin{equation*} \bigcup_{X \in \mathscr{C}}^{}X = \left\{x \in U \mid x \in X \text{ for some } X \in \mathscr{C} \right\} \end{equation*}

The intersection of \(\mathbf{\mathcal{C}}\) is defined as the set of all elements that are in all of the sets in \(\mathcal{C}\text{.}\) That is,

\begin{equation*} \bigcap_{X \in \mathscr{C}}^{}X = \left\{x \in U \mid x \in X \text{ for all } X \in \mathscr{C} \right\} \end{equation*}

For example, consider the four sets \(A\text{,}\) \(B\text{,}\) \(C\text{,}\) and \(D\) used earlier in this activity and the sets

\begin{equation*} S = \left\{5, 6, 7, 8, 9 \right\} \text{ and } T = \left\{6, 7, 8, 9, 10 \right\}\text{.} \end{equation*}

We can then consider the following families of sets: \(\mathscr{A} = \left\{A, B, C , D \right\}\) and \(\mathscr{B} = \left\{A, B, C , D, S, T \right\}\text{.}\)

We often use subscripts to identify sets. For example, in Beginning Activity 1, instead of using \(A\text{,}\) \(B\text{,}\) \(C\text{,}\) and \(D\) as the names of the sets, we could have used \(A_1\text{,}\) \(A_2\text{,}\) \(A_3\text{,}\) and \(A_4\text{.}\) When we do this, we are using the subscript as an identifying tag, or index, for each set. We can also use this idea to specify an infinite family of sets. For example, for each natural number \(n\text{,}\) we define

\begin{equation*} C_n = \left\{ n, n+1, n+2, n+3, n+4 \right\}\!\text{.} \end{equation*}

So if we have a family of sets \(\mathscr{C} = \left\{ C_1, C_2, C_3, C_4 \right\}\text{,}\) we use the notation \(\bigcup\limits_{j=1}^{4}C_j\) to mean the same thing as \(\bigcup\limits_{X \in \mathscr{C}}^{}X\text{.}\)

Subsection The Union and Intersection of an Indexed Family of Sets

One of the purposes of the beginning activities was to show that we often encounter situations in which more than two sets are involved, and it is possible to define the union and intersection of more than two sets. In Beginning Activity 2, we also saw that it is often convenient to “index” the sets in a family of sets. In particular, if \(n\) is a natural number and \(\mathscr{A} = \left\{ A_1, A_2, \ldots , A_n \right\}\) is a family of \(n\) sets, then the union of these \(n\) sets, denoted by \(A_1 \cup A_2 \cup \cdots \cup A_n\) or \(\bigcup\limits_{j=1}^{n}A_j\text{,}\) is defined as

\begin{equation*} \bigcup_{j=1}^{n}A_j = \left\{ x \in U \mid x \in A_j, \text{ for some } j \text{ with } 1 \leq j \leq n \right\}\!\text{.} \end{equation*}

We can also define the intersection of these \(n\) sets, denoted by \(A_1 \cap A_2 \cap \cdots \cap A_n\) or \(\bigcap\limits_{j=1}^{n}A_j\text{,}\) as

\begin{equation*} \bigcap_{j=1}^{n}A_j = \left\{ x \in U \mid x \in A_j, \text{ for all } j \text{ with } 1 \leq j \leq n \right\}\!\text{.} \end{equation*}

We can also extend this idea to define the union and intersection of a family that consists of infinitely many sets. So if \(\mathscr{B} = \left\{ B_1, B_2, \ldots , B_n, \ldots \: \right\}\text{,}\) then

\begin{align*} \bigcup_{j=1}^{\infty}B_j \amp = \left\{ x \in U \mid x \in B_j, \text{ for some } j \text{ with } j \geq 1 \right\}\!, \text{ and }\\ \bigcap_{j=1}^{\infty}B_j \amp = \left\{ x \in U \mid x \in B_j, \text{ for all } j \text{ with } j \geq 1 \right\}\!. \end{align*}

In all of the examples we have studied so far, we have used \(\N\) or a subset of \(\N\) to index or label the sets in a family of sets. We can use other sets to index or label sets in a family of sets. For example, for each real number \(x\text{,}\) we can define \(B_x\) to be the closed interval \(\left[x, x + 2 \right]\text{.}\) That is,

\begin{equation*} B_x = \left\{ y \in \R \mid x \leq y \leq x + 2 \right\}\text{.} \end{equation*}

So we make the following definition. In this definition, \(\Lambda\) is the uppercase Greek letter lambda and \(\alpha\) is the lowercase Greek letter alpha.

Definition.

Let \(\Lambda\) be a nonempty set and suppose that for each \(\alpha \in \Lambda\text{,}\) there is a corresponding set \(A_\alpha\text{.}\) The family of sets \(\left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) is called an indexed family of sets indexed by \(\Lambda\text{.}\) Each \(\alpha \in \Lambda\) is called an index and \(\Lambda\) is called an indexing set.

Progress Check 5.36. Indexed Families of Sets.

We now restate the definitions of the union and intersection of a family of sets for an indexed family of sets.

Definition.

Let \(\Lambda\) be a nonempty indexing set and let \(\mathscr{A} = \left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) be an indexed family of sets. The union over \(\mathbf{\mathscr{A}}\) is defined as the set of all elements that are in at least one of sets \(A_\alpha\text{,}\) where \(\alpha \in \Lambda\text{.}\) We write

\begin{equation*} \bigcup_{\alpha \in \Lambda}^{}A_\alpha = \left\{ x \in U \mid \text{ there exists an } \alpha \in \Lambda \text{ with } x \in A_\alpha \right\}\!\text{.} \end{equation*}

The intersection over \(\mathbf{\mathscr{A}}\) is the set of all elements that are in all of the sets \(A_\alpha\) for each \(\alpha \in \Lambda\text{.}\) That is,

\begin{equation*} \bigcap_{\alpha \in \Lambda}^{}A_\alpha = \left\{ x \in U \mid \text{ for all } \alpha \in \Lambda, x \in A_\alpha \right\}\!\text{.} \end{equation*}
Example 5.37. A Family of Sets Indexed by the Positive Real Numbers.

For each positive real number \(\alpha\text{,}\) let \(A_\alpha\) be the interval \(\left( -1, \alpha \right]\text{.}\) That is,

\begin{equation*} A_\alpha = \left\{ x \in \R \mid -1 \lt x \leq \alpha \right\}\!\text{.} \end{equation*}

If we let \(\R^+\) be the set of positive real numbers, then we have a family of sets indexed by \(\R^+\text{.}\) We will first determine the union of this family of sets. Notice that for each \(\alpha \in \R^+\text{,}\) \(\alpha \in A_\alpha\text{,}\) and if \(y\) is a real number with \(-1 \lt y \leq 0\text{,}\) then \(y \in A_\alpha\text{.}\) Also notice that if \(y \in \R\) and \(y \lt -1\text{,}\) then for each \(\alpha \in \R^+\text{,}\) \(y \notin A_\alpha\text{.}\) With these observations, we conclude that

\begin{equation*} \bigcup_{\alpha \in \R^+}^{}A_\alpha = \left( -1, \infty \right) = \left\{ x \in \R \mid -1 \lt x \right\}\!\text{.} \end{equation*}

To determine the intersection of this family, notice that

  • if \(y \in \R\) and \(y \lt -1\text{,}\) then for each \(\alpha \in \R^+\text{,}\) \(y \notin A_\alpha\text{;}\)

  • if \(y \in \R\) and \(-1 \lt y \leq 0\text{,}\) then \(y \in A_\alpha\) for each \(\alpha \in \R^+\text{;}\) and

  • if \(y \in \R\) and \(y > 0\text{,}\) then if we let \(\beta = \dfrac{y}{2}\text{,}\) \(y > \beta\) and \(y \notin A_\beta\text{.}\)

From these observations, we conclude that

\begin{equation*} \bigcap_{\alpha \in \R^+}^{}A_\alpha = \left( -1, 0 \right] = \left\{ x \in \R \mid -1 \lt x \leq 0 \right\}\!\text{.} \end{equation*}

Subsection Properties of Union and Intersection

In Theorem 5.39, we will prove some properties of set operations for indexed families of sets. Some of these properties are direct extensions of corresponding properties for two sets. For example, we have already proved De Morgan's Laws for two sets in Theorem 5.26. The work in the beginning activities and Progress Check 5.38 suggests that we should get similar results using set operations with an indexed family of sets. For example, in Beginning Activity 2, we saw that

\begin{equation*} \left( \bigcap_{j=1}^{4}A_j \right)^c = \bigcup_{j=1}^{4}A_j^c\text{.} \end{equation*}
Theorem 5.39.

Let \(\Lambda\) be a nonempty indexing set and let \(\mathscr{A} = \left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) be an indexed family of sets, each of which is a subset of some universal set \(U\text{.}\) Then

  1. For each \(\beta \in \Lambda\text{,}\) \(\bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha \subseteq A_\beta\text{.}\)

  2. For each \(\beta \in \Lambda\text{,}\) \(A_\beta \subseteq \bigcup\limits_{\alpha \in \Lambda}^{}A_\alpha\text{.}\)

  3. \(\displaystyle \left(\:\bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha \right)^c = \bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha}^c\)

  4. \(\displaystyle \left(\:\bigcup\limits_{\alpha \in \Lambda}^{}A_\alpha \right)^c = \bigcap\limits_{\alpha \in \Lambda}^{}A_{\alpha}^c\)

Item 3 and Item 4 are known as De Morgan's Laws.

Proof.

We will prove Item 1 and Item 3. The proofs of Item 2 and Item 4 are included in Exercise 4. So we let \(\Lambda\) be a nonempty indexing set and let \(\mathscr{A} = \left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) be an indexed family of sets. To prove Item 1, we let \(\beta \in \Lambda\) and note that if \(x \in \bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha\text{,}\) then \(x \in A_\alpha\text{,}\) for all \(\alpha \in \Lambda\text{.}\) Since \(\beta\) is one element in \(\Lambda\text{,}\) we may conclude that \(x \in A_\beta\text{.}\) This proves that \(\bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha \subseteq A_\beta\text{.}\)

To prove Item 3, we will prove that each set is a subset of the other set. We first let \(x \in \left(\:\bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha \right)^c\text{.}\) This means that \(x \notin \left(\:\bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha \right)\text{,}\) and this means that

there exists a \(\beta \in \Lambda\) such that \(x \notin A_\beta\text{.}\)
Hence, \(x \in A_{\beta}^c\text{,}\) which implies that \(x \in \bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha}^c\text{.}\) Therefore, we have proved that

\begin{equation*} \left(\:\bigcap_{\alpha \in \Lambda}^{}A_\alpha \right)^c \subseteq \bigcup_{\alpha \in \Lambda}^{}A_{\alpha}^c \qquad (1)\text{.} \end{equation*}

We now let \(y \in \bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha}^c\text{.}\) This means that there exists a \(\beta \in \Lambda\) such that \(y \in A_{\beta}^c\) or \(y \notin A_\beta\text{.}\) However, since \(y \notin A_\beta\text{,}\) we may conclude that \(y \notin \bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha\) and, hence, \(y \in \left(\:\bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha \right)^c\text{.}\) This proves that

\begin{equation*} \bigcup_{\alpha \in \Lambda}^{}A_{\alpha}^c \subseteq \left(\:\bigcap_{\alpha \in \Lambda}^{}A_\alpha \right)^c \qquad (2)\text{.} \end{equation*}

Using the results in (1) and (2), we have proved that \(\left(\:\bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha \right)^c = \bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha}^c\text{.}\)

Many of the other properties of set operations are also true for indexed families of sets. Theorem 5.40 states the distributive laws for set operations.

Theorem 5.40.

Let \(\Lambda\) be a nonempty indexing set, let \(\mathscr{A} = \left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) be an indexed family of sets, and let \(B\) be a set. Then

  1. \(B \cap \left(\:\bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right) = \bigcup\limits_{\alpha \in \Lambda}^{} \left( B \cap A_{\alpha} \right)\text{,}\) and

  2. \(B \cup \left(\:\bigcap\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right) = \bigcap\limits_{\alpha \in \Lambda}^{} \left( B \cup A_{\alpha} \right)\text{.}\)

The proof of Theorem 5.40 is Exercise 5.

Subsection Pairwise Disjoint Families of Sets

In Section 5.2, we defined two sets \(A\) and \(B\) to be disjoint provided that \(A \cap B = \emptyset\text{.}\) In a similar manner, if \(\Lambda\) is a nonempty indexing set and \(\mathscr{A} = \left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) is an indexed family of sets, we can say that this indexed family of sets is disjoint provided that \(\bigcap\limits_{\alpha \in \Lambda}^{}A_\alpha = \emptyset\text{.}\) However, we can use the concept of two disjoint sets to define a somewhat more interesting type of “disjointness” for an indexed family of sets.

Definition.

Let \(\Lambda\) be a nonempty indexing set, and let \(\mathscr{A} = \left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) be an indexed family of sets. We say that \(\mathscr{A}\) is pairwise disjoint provided that for all \(\alpha\) and \(\beta\) in \(\Lambda\text{,}\) if \(A_\alpha \ne A_\beta\text{,}\) then \(A_\alpha \cap A_\beta = \emptyset\text{.}\)

Progress Check 5.41. Disjoint Families of Sets.

Figure 5.42 shows two families of sets,

\begin{equation*} \mathscr{A} = \left\{A_1, A_2, A_3, A_4 \right\} \text{ and } \mathscr{B} = \left\{B_1, B_2, B_3, B_4 \right\} \end{equation*}
(a)

Is the family of sets \(\mathscr{A}\) a disjoint family of sets? A pairwise disjoint family of sets?

(b)

Is the family of sets \(\mathscr{B}\) a disjoint family of sets? A pairwise disjoint family of sets?

(c)

Now let the universal set be \(\R\text{.}\) For each \(n \in \N\text{,}\) let \(C_n = \left( n, \infty \right)\text{,}\) and let \(\mathscr{C} = \left\{ C_n \mid n \in \N \right\}\text{.}\)

Is the family of sets \(\mathscr{C}\) a disjoint family of sets? A pairwise disjoint family of sets?

Solution.

All three families of sets \(\left( \mathscr{A}, \mathscr{B}, \text{ and } \mathscr{C} \right)\) are disjoint families of sets. Only the family \(\mathscr{A}\) is a pairwise disjoint family of sets.

Prove Theorem 5.40. Let \(\Lambda\) be a nonempty indexing set, let \(\mathscr{A} = \left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) be an indexed family of sets, and let \(B\) be a set. Then

(a)

\(B \cap \left(\:\bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right) = \bigcup\limits_{\alpha \in \Lambda}^{} \left( B \cap A_{\alpha} \right)\text{,}\) and

Answer.

We first let \(x \in B \cap \left(\:\bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right)\text{.}\) Then \(x \in B\) and \(x \in \bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha}\text{.}\) This means that there exists an \(\alpha \in \Lambda\) such that \(x \in A_\alpha\text{.}\) Hence, \(x \in B \cap A_\alpha\text{,}\) which implies that \(x \in \bigcup\limits_{\alpha \in \Lambda}^{} \left( B \cap A_{\alpha} \right)\text{.}\) This proves that \(B \cap \left(\bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right) \subseteq \bigcup\limits_{\alpha \in \Lambda}^{} \left( B \cap A_{\alpha} \right)\text{.}\)

We now let \(y \in \bigcup\limits_{\alpha \in \Lambda}^{} \left( B \cap A_{\alpha} \right)\text{.}\) So there exists an \(\alpha \in \Lambda\) such that \(y \in B \cap A_{\alpha}\text{.}\) Then \(y \in B\) and \(y \in A_{\alpha}\text{,}\) which implies that \(y \in B\) and \(y \in \bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha}\text{.}\) Therefore, \(y \in B \cap \left(\:\bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right)\text{,}\) and this proves that \(\bigcup\limits_{\alpha \in \Lambda}^{} \left( B \cap A_{\alpha} \right) \subseteq B \cap \left(\bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right)\text{.}\)

(b)

\(B \cup \left(\:\bigcap\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right) = \bigcap\limits_{\alpha \in \Lambda}^{} \left( B \cup A_{\alpha} \right)\text{.}\)

12.

Let \(\Lambda\) be a nonempty indexing set, let \(\mathscr{A} = \left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) be an indexed family of sets, and let \(B\) be a set. Use the results of Theorem 5.39 and Theorem 5.40 to prove each of the following:

(a)

\(\left(\:\bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right) - B = \bigcup\limits_{\alpha \in \Lambda}^{} \left( A_{\alpha} - B \right)\)

Answer.

We first rewrite the set difference and then use a distributive law.

\begin{align*} \left(\bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right) - B \amp = \left(\bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right) \cap B^c\\ \amp = \bigcup\limits_{\alpha \in \Lambda}^{}\left( A_\alpha \cap B^c \right)\\ \amp =\bigcup\limits_{\alpha \in \Lambda}^{} \left( A_{\alpha} - B \right) \end{align*}
(b)

\(\left(\:\bigcap\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right) - B = \bigcap\limits_{\alpha \in \Lambda}^{} \left( A_{\alpha} - B \right)\)

(c)

\(B - \left(\:\bigcup\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right) = \bigcap\limits_{\alpha \in \Lambda}^{} \left( B - A_{\alpha} \right)\)

(d)

\(B - \left(\:\bigcap\limits_{\alpha \in \Lambda}^{}A_{\alpha} \right) = \bigcup\limits_{\alpha \in \Lambda}^{} \left( B - A_{\alpha} \right)\)
