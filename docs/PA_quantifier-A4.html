Beginning Activity Beginning Activity 1: An Introduction to Quantifiers
A4US

We have seen that one way to create a statement from an open sentence is to substitute a specific element from the universal set for each variable in the open sentence. Another way is to make some claim about the truth set of the open sentence. This is often done by using a quantifier. For example, if the universal set is \(\mathbb{R}\text{,}\) then the following sentence is a statement.

For each real number \(x\text{,}\) \(x^2 > 0\text{.}\)
The phrase âFor each real number \(x\)â is said to quantify the variable that follows it in the sense that the sentence is claiming that something is true for all real numbers. So this sentence is a statement (which happens to be false).

Definition.

The phrase âfor everyâ (or its equivalents) is called a universal quantifier. The phrase âthere existsâ (or its equivalents) is called an existential quantifier. The symbol \(\forall\) is used to denote a universal quantifier, and the symbol \(\exists\) is used to denote an existential quantifier.

Using this notation, the statement âFor each real number \(x\text{,}\) \(x^2 > 0\)â could be written in symbolic form as: \(\left( {\forall x \in \mathbb{R}} \right)\left( {x^2 > 0} \right)\text{.}\) The following is an example of a statement involving an existential quantifier.

There exists an integer \(x\) such that \(3x - 2 = 0\text{.}\)
This could be written in symbolic form as

\begin{equation*} \left( {\exists x \in \Z} \right) \left( 3x - 2 = 0 \right)\text{.} \end{equation*}

This statement is false because there are no integers that are solutions of the linear equation \(3x - 2 = 0\text{.}\) TableÂ 1.21 summarizes the facts about the two types of quantifiers.

In effect, the table indicates that the universally quantified statement is true provided that the truth set of the predicate equals the universal set, and the existentially quantified statement is true provided that the truth set of the predicate contains at least one element.
