Section 2.4 Quantifiers and Negations

Worksheet Beginning Activity 1: An Introduction to Quantifiers
A4US

We have seen that one way to create a statement from an open sentence is to substitute a specific element from the universal set for each variable in the open sentence. Another way is to make some claim about the truth set of the open sentence. This is often done by using a quantifier. For example, if the universal set is \(\mathbb{R}\text{,}\) then the following sentence is a statement.

For each real number \(x\text{,}\) \(x^2 > 0\text{.}\)
The phrase âFor each real number \(x\)â is said to quantify the variable that follows it in the sense that the sentence is claiming that something is true for all real numbers. So this sentence is a statement (which happens to be false).
Definition 2.4.1.

The phrase âfor everyâ (or its equivalents) is called a universal quantifier. The phrase âthere existsâ (or its equivalents) is called an existential quantifier. The symbol \(\forall\) is used to denote a universal quantifier, and the symbol \(\exists\) is used to denote an existential quantifier.

Using this notation, the statement âFor each real number \(x\text{,}\) \(x^2 > 0\)â could be written in symbolic form as: \(\left( {\forall x \in \mathbb{R}} \right)\left( {x^2 > 0} \right)\text{.}\) The following is an example of a statement involving an existential quantifier.

There exists an integer \(x\) such that \(3x - 2 = 0\text{.}\)
This could be written in symbolic form as

\begin{equation*} \left( {\exists x \in \Z} \right) \left( 3x - 2 = 0 \right)\text{.} \end{equation*}

This statement is false because there are no integers that are solutions of the linear equation \(3x - 2 = 0\text{.}\) TableÂ 2.4.2 summarizes the facts about the two types of quantifiers.

In effect, the table indicates that the universally quantified statement is true provided that the truth set of the predicate equals the universal set, and the existentially quantified statement is true provided that the truth set of the predicate contains at least one element.

Subsection Forms of Quantified Statements in English

There are many ways to write statements involving quantifiers in English. In some cases, the quantifiers are not apparent, and this often happens with conditional statements. The following examples illustrate these points. Each example contains a quantified statement written in symbolic form followed by several ways to write the statement in English.

Example 2.4.3.
\begin{equation*} \left( {\forall x \in \mathbb{R}} \right)\left( {x^2 > 0} \right) \end{equation*}

  • For each real number \(x\text{,}\) \(x^2 > 0\text{.}\)

  • The square of every real number is greater than 0.

  • The square of a real number is greater than 0.

  • If \(x \in \mathbb{R}\text{,}\) then \(x^2 > 0\text{.}\)

In the second to the last example, the quantifier is not stated explicitly. Care must be taken when reading this because it really does say the same thing as the previous examples. The last example illustrates the fact that conditional statements often contain a âhiddenâ universal quantifier. If the universal set is \(\R\text{,}\) then the truth set of the open sentence \(x^2 > 0\) is the set of all nonzero real numbers. That is, the truth set is

\begin{equation*} \left\{ {x \in \mathbb{R}} \mid x \ne 0 \right\}\text{.} \end{equation*}

So the preceding statements are false. For the conditional statement, the example using \(x = 0\) produces a true hypothesis and a false conclusion. This is a counterexample that shows that the statement with a universal quantifier is false.

Example 2.4.4.
\begin{equation*} \left( {\exists x \in \mathbb{R}} \right)\left( {x^2 = 5} \right) \end{equation*}

  • There exists a real number \(x\) such that \(x^2 = 5\text{.}\)

  • \(x^2 = 5\) for some real number \(x\text{.}\)

  • There is a real number whose square equals 5.

The second example is usually not used since it is not considered good writing practice to start a sentence with a mathematical symbol. If the universal set is \(\R\text{,}\) then the truth set of the predicate â\(x^2 = 5\)â is \(\left\{ { - \sqrt 5 ,\;\sqrt 5 } \right\}\text{.}\) So these are all true statements.

Subsection Negations of Quantified Statements

In Beginning Activity 1Â , we wrote negations of some quantified statements. This is a very important mathematical activity. As we will see in future sections, it is sometimes just as important to be able to describe when some object does not satisfy a certain property as it is to describe when the object satisfies the property. Our next task is to learn how to write negations of quantified statements in a useful English form.

We first look at the negation of a statement involving a universal quantifier. The general form for such a statement can be written as \(\left( {\forall x} \in U \right)\left( {P( x )} \right)\text{,}\) where \(P( x )\) is an open sentence and \(U\) is the universal set for the variable \(x\text{.}\) When we write

\begin{equation*} \mynot \left( {\forall x} \in U \right)\left[ {P\left( x \right)} \right]\text{,} \end{equation*}

we are asserting that the statement \(\left( {\forall x} \in U \right)\left[ {P( x )} \right]\) is false. This is equivalent to saying that the truth set of the open sentence \(P( x )\) is not the universal set. That is, there exists an element \(x\) in the universal set \(U\) such that \(P( x )\) is false. This in turn means that there exists an element \(x\) in \(U\) such that \(\mynot P( x )\) is true, which is equivalent to saying that \(\left( {\exists x} \in U \right)\left[ {\mynot P( x )} \right]\) is true. This explains why the following result is true:

\begin{equation*} \mynot \left( {\forall x} \in U \right)\left[ {P( x )} \right] \equiv \left( {\exists x} \in U \right)\left[ {\mynot P( x )} \right]\text{.} \end{equation*}

Similarly, when we write

\begin{equation*} \mynot \left( {\exists x} \in U \right)\left[ {P( x )} \right]\text{,} \end{equation*}

we are asserting that the statement \(\left( {\exists x} \in U \right)\left[ {P( x )} \right]\) is false. This is equivalent to saying that the truth set of the open sentence \(P( x )\) is the empty set. That is, there is no element \(x\) in the universal set \(U\) such that \(P( x )\) is true. This in turn means that for each element \(x\) in \(U\text{,}\) \(\mynot P( x )\) is true, and this is equivalent to saying that \(\left( {\forall x} \in U \right)\left[ {\mynot P( x )} \right]\) is true. This explains why the following result is true:

\begin{equation*} \mynot \left( {\exists x} \in U \right)\left[ {P( x )} \right] \equiv \left( {\forall x} \in U \right)\left[ {\mynot P( x )} \right]\text{.} \end{equation*}

We summarize these results in the following theorem.

Theorem 2.4.5.

For any open sentence \(P( x )\text{,}\)

\begin{align*} \mynot \left( {\forall x} \in U \right)\left[ {P( x )} \right] \amp \equiv \left( {\exists x} \in U \right)\left[ {\mynot P( x )} \right]\text{ , and }\\ \mynot \left( {\exists x} \in U \right)\left[ {P( x )} \right] \amp \equiv \left( {\forall x} \in U \right)\left[ {\mynot P( x )} \right]. \end{align*}
Example 2.4.6. Negations of Quantified Statements.

Consider the following statement: \(\left( {\forall x \in \mathbb{R}} \right)\left( {x^3 \geq x^2 } \right)\text{.}\)

We can write this statement as an English sentence in several ways. Following are two different ways to do so.

  • For each real number \(x\text{,}\) \(x^3 \geq x^2\text{.}\)

  • If \(x\) is a real number, then \(x^3\) is greater than or equal to \(x^2\text{.}\)

The second statement shows that in a conditional statement, there is often a hidden universal quantifier. This statement is false since there are real numbers \(x\) for which \(x^3\) is not greater than or equal to \(x^2\text{.}\) For example, we could use \(x = - 1\) or \(x = \frac{1}{2}\text{.}\)

This means that the negation must be true. We can form the negation as follows:

\begin{equation*} \mynot \left( {\forall x \in \mathbb{R}} \right)\left( {x^3 \geq x^2 } \right) \equiv \left( {\exists x \in \mathbb{R}} \right)\mynot \left( {x^3 \geq x^2 } \right)\text{.} \end{equation*}

In most cases, we want to write this negation in a way that does not use the negation symbol. In this case, we can now write the open sentence \(\mynot \left( {x^3 \geq x^2 } \right)\) as \(\left( {x^3 \lt x^2 } \right)\text{.}\) (That is, the negation of âis greater than or equal toâ is âis less than.â) So we obtain the following:

\begin{equation*} \mynot \left( {\forall x \in \mathbb{R}} \right)\left( {x^3 \geq x^2 } \right) \equiv \left( {\exists x \in \mathbb{R}} \right)\left( {x^3 \lt x^2 } \right)\text{.} \end{equation*}

The statement \(\left( {\exists x \in \mathbb{R}} \right)\left( {x^3 \lt x^2 } \right)\) could be written in English as follows:

There exists a real number \(x\) such that \(x^3 \lt x^2\text{.}\)
There exists an \(x\) such that \(x\) is a real number and \(x^3 \lt x^2\text{.}\)

  2. Write the negation of the statement in a symbolic form that does not use the negation symbol.

  3. Write the negation of the statement in the form of an English sentence that does not use the symbols for quantifiers.

\(\left( \forall a \in \mathbb{R}\right) \left( a + 0 = a \right)\text{.}\)

Solution.

  • For each real number \(x\text{,}\) \(\sin ( {2x} ) = 2( {\sin x} )( {\cos x} )\text{.}\)

  • \((\exists x \in \mathbb{R})( \sin ( {2x} ) \ne 2( {\sin x} )( {\cos x} ))\text{.}\)

  • There exists a real number \(x\) such that \(\sin \left( {2x} \right) \ne 2\left( {\sin x} \right)\left( {\cos x} \right)\text{.}\)

(c)

\(\left( \forall x \in \mathbb{R} \right) \left( \tan ^2 x + 1 = \sec ^2 x \right)\text{.}\)

Solution.

  • For each real number \(x\text{,}\) \(\tan ^2 x + 1 = \sec ^2 x\text{.}\)

  • \(\left( \exists x \in \mathbb{R}\right) \left( \tan ^2 x + 1 \ne \sec ^2 x \right)\text{.}\)

  • There exists a real number \(x\) such that \(\tan ^2 x + 1 \ne \sec ^2 x\text{.}\)

(d)

\(\left( \exists x \in \mathbb{Q} \right) \left( x^2 - 3x - 7 = 0 \right)\text{.}\)

Solution.

  • There exists a rational number \(x\) such that \(x^2 - 3x - 7 = 0\text{.}\)

  • \((\forall x \in \mathbb{Q}) \left( x^2 - 3x - 7 \ne 0 \right)\text{.}\)

  • For each rational number \(x\text{,}\) \(x^2 - 3x - 7 \ne 0\text{.}\)

(e)

\(\left( \exists x \in \mathbb{R} \right) \left( x^2 + 1 = 0 \right)\text{.}\)

Solution.

  • There exists a real number \(x\) such that \(x^2 + 1 = 0\text{.}\)

  • \((\forall x \in \mathbb{R}) \left( \text{ } x^2 + 1 \ne 0 \right)\text{.}\)

  • For each real number \(x\text{,}\) \(x^2 + 1 \ne 0\text{.}\)

Subsection Counterexamples and Negations of Conditional Statements

The real number \(x = - 1\) in the previous example was used to show that the statement \(\left( {\forall x \in \mathbb{R}} \right)\left( {x^3 \geq x^2 } \right)\) is false. This is called a counterexample to the statement. In general, a counterexample to a statement of the form \(\left( {\forall x} \right)\left[ {P( x )} \right]\) is an object \(a\) in the universal set \(U\) for which \(P( a )\) is false. It is an example that proves that \(\left( {\forall x} \right)\left[ {P( x )} \right]\) is a false statement, and hence its negation, \(\left( {\exists x} \right)\left[ {\mynot P( x )} \right]\text{,}\) is a true statement.

In the preceding example, we also wrote the universally quantified statement as a conditional statement. The number \(x = - 1\) is a counterexample for the statement If \(x\) is a real number, then \(x^3\) is greater than or equal to \(x^2\text{.}\) So the number \(-1\) is an example that makes the hypothesis of the conditional statement true and the conclusion false. Remember that a conditional statement often contains a âhiddenâ universal quantifier. Also, recall that in SectionÂ 2.2 we saw that the negation of the conditional statement âIf \(P\) then \(Q\)â is the statement â\(P\) and not \(Q\text{.}\)â Symbolically, this can be written as follows:

\begin{equation*} \mynot \left( {P \to Q} \right) \equiv \;P \wedge \mynot Q\text{.} \end{equation*}

So when we specifically include the universal quantifier, the symbolic form of the negation of a conditional statement is

\begin{align*} \mynot \left( {\forall x} \in U \right)\left[ {P( x ) \to Q( x )} \right] \amp \equiv \left( {\exists x} \in U \right)\mynot \left[ {P( x ) \to Q( x )} \right]\\ \amp \equiv \left( {\exists x} \in U \right)\left[ {P( x ) \wedge \mynot Q( x )} \right] \end{align*}

That is,

\begin{equation*} \mynot \left( {\forall x} \in U \right)\left[ {P( x ) \to Q( x )} \right] \equiv \left( {\exists x} \in U \right)\left[ {P( x ) \wedge \mynot Q( x )} \right]\text{.} \end{equation*}
Progress Check 2.4.8. Using Counterexamples.

Use counterexamples to explain why each of the following statements is false.

(a)

For each integer \(n\text{,}\) \(\left( n^2 + n + 1 \right)\) is a prime number.

Solution.

A counterexample is \(n = 4\) since \(4^2 + 4 + 1 = 21\text{,}\) and 21 is not prime.

(b)

For each real number \(x\text{,}\) if \(x\) is positive, then \(2x^2 > x\text{.}\)

Solution.

A counterexample is \(x = \dfrac{1}{4}\) since \(\dfrac{1}{4}\) is positive and \(2 \left( \dfrac{1}{4} \right)^2 = \dfrac{1}{8}\) and \(\dfrac{1}{8} \leq \dfrac{1}{4}\text{.}\)

Subsection Quantifiers in Definitions

Definitions of terms in mathematics often involve quantifiers. These definitions are often given in a form that does not use the symbols for quantifiers. Not only is it important to know a definition, it is also important to be able to write a negation of the definition. This will be illustrated with the definition of what it means to say that a natural number is a perfect square.

Definition 2.4.9.

A natural number \(n\) is a perfect square provided that there exists a natural number \(k\) such that \(n = k^2\text{.}\)

This definition can be written in symbolic form using appropriate quantifiers as follows:

A natural number \(n\) is a perfect square provided \(\left( {\exists k \in \mathbb{N}} \right) \! \left( {n = k^2 } \right)\text{.}\)

We frequently use the following steps to gain a better understanding of a definition.

  1. Examples of natural numbers that are perfect squares are 1, 4, 9, and 81 since \(1 = 1^2\text{,}\) \(4 = 2^2\text{,}\) \(9 = 3^2\text{,}\) and \(81 = 9^2\text{.}\)

  2. Examples of natural numbers that are not perfect squares are 2, 5, 10, and 50.

  3. This definition gives two âconditions.â One is that the natural number \(n\) is a perfect square and the other is that there exists a natural number \(k\) such that \(n = k^2\text{.}\) The definition states that these mean the same thing. So when we say that a natural number \(n\) is not a perfect square, we need to negate the condition that there exists a natural number \(k\) such that \(n = k^2\text{.}\) We can use the symbolic form to do this.

    \begin{equation*} \mynot \left( {\exists k \in \mathbb{N}} \right)\left( {n = k^2 } \right) \equiv \left( \forall k \in \N \right) \left( n \ne k^2 \right) \end{equation*}

    Notice that instead of writing \(\mynot \left(n = k^2 \right)\text{,}\) we used the equivalent form of \(\left(n \ne k^2 \right)\text{.}\) This will be easier to translate into an English sentence. So we can write,

    A natural number \(n\) is not a perfect square provided that for every natural number \(k\text{,}\) \(n \ne k^2\text{.}\)

The preceding method illustrates a good method for trying to understand a new definition. Most textbooks will simply define a concept and leave it to the reader to do the preceding steps. Frequently, it is not sufficient just to read a definition and expect to understand the new term. We must provide examples that satisfy the definition, as well as examples that do not satisfy the definition, and we must be able to write a coherent negation of the definition.

Subsection Statements with More than One Quantifier

When a predicate contains more than one variable, each variable must be quantified to create a statement. For example, assume the universal set is the set of integers, \(\mathbb{Z}\text{,}\) and let \(P\left( {x, y} \right)\) be the predicate, â\(x + y = 0\text{.}\)â We can create a statement from this predicate in several ways.

  1. \(\left( {\forall x \in \mathbb{Z}} \right)\left( {\forall y \in \mathbb{Z}} \right) \left( {x + y = 0} \right)\)

    We could read this as, âFor all integers \(x\) and \(y\text{,}\) \(x + y = 0\text{.}\)â This is a false statement since it is possible to find two integers whose sum is not zero \(\left( {2 + 3 \ne 0} \right)\text{.}\)

  2. \(\left( {\forall x \in \mathbb{Z}} \right)\left( {\exists y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\)

    We could read this as, âFor every integer \(x\text{,}\) there exists an integer \(y\) such that \(x + y = 0\text{.}\)â This is a true statement.

  3. \(\left( {\exists x \in \mathbb{Z}} \right)\left( {\forall y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\)

    We could read this as, âThere exists an integer \(x\) such that for each integer \(y\text{,}\) \(x + y = 0\text{.}\)â This is a false statement since there is no integer whose sum with each integer is zero.

  4. \(\left( {\exists x \in \mathbb{Z}} \right)\left( {\exists y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\)

    We could read this as, âThere exist integers \(x\) and \(y\) such that \(x + y = 0\text{.}\)â This is a true statement. For example, \(2 + \left( { - 2} \right) = 0\text{.}\)

When we negate a statement with more than one quantifier, we consider each quantifier in turn and apply the appropriate part of TheoremÂ 2.4.5. As an example, we will negate ItemÂ 3 from the preceding list. The statement is

\begin{equation*} \left( {\exists x \in \mathbb{Z}} \right)\left( {\forall y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\text{.} \end{equation*}

We first treat this as a statement in the following form: \(\left( {\exists x \in \mathbb{Z}} \right)\left( {P( x )} \right)\) where \(P( x )\) is the predicate \(\left( {\forall y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\text{.}\) Using TheoremÂ 2.4.5, we have

\begin{equation*} \mynot \left( {\exists x \in \mathbb{Z}} \right)\left( {P( x )} \right) \equiv \left( {\forall x \in \mathbb{Z}} \right)\left( {\mynot P( x )} \right)\text{.} \end{equation*}

Using TheoremÂ 2.4.5 again, we obtain the following:

\begin{align*} \mynot P( x ) \amp \equiv \mynot \left( {\forall y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\\ \amp \equiv \left( {\exists y \in \mathbb{Z}} \right)\mynot \left( {x + y = 0} \right)\\ \amp \equiv \left( {\exists y \in \mathbb{Z}} \right)\left( {x + y \ne 0} \right) \end{align*}

Combining these two results, we obtain

\begin{equation*} \mynot \left( {\exists x \in \mathbb{Z}} \right)\left( {\forall y \in \mathbb{Z}} \right)\left( {x + y = 0} \right) \equiv \left( {\forall x \in \mathbb{Z}} \right)\left( {\exists y \in \mathbb{Z}} \right)\left( {x + y \ne 0} \right)\text{.} \end{equation*}

Subsection Writing Guideline

Try to use English and minimize the use of cumbersome notation. Do not use the special symbols for quantifiers \(\forall\) (for all), \(\exists\) (there exists), \(\mathrel\backepsilon\) (such that), or \(\therefore\) (therefore) in formal mathematical writing. It is often easier to write and usually easier to read, if the English words are used instead of the symbols. For example, why make the reader interpret

\begin{equation*} \left( \forall x \in \R \right) \left( \exists y \in \R \right)\left( x + y = 0 \right) \end{equation*}

when it is possible to write

For each real number \(x\text{,}\) there exists a real number \(y\) such that \(x + y = 0\text{,}\)
or, more succinctly (if appropriate),
Every real number has an additive inverse.
