Beginning Activity Beginning Activity 2: An Indexed Family of Sets
A4US

We often use subscripts to identify sets. For example, in Beginning Activity 1, instead of using \(A\text{,}\) \(B\text{,}\) \(C\text{,}\) and \(D\) as the names of the sets, we could have used \(A_1\text{,}\) \(A_2\text{,}\) \(A_3\text{,}\) and \(A_4\text{.}\) When we do this, we are using the subscript as an identifying tag, or index, for each set. We can also use this idea to specify an infinite family of sets. For example, for each natural number \(n\text{,}\) we define

\begin{equation*} C_n = \left\{ n, n+1, n+2, n+3, n+4 \right\}\!\text{.} \end{equation*}

So if we have a family of sets \(\mathscr{C} = \left\{ C_1, C_2, C_3, C_4 \right\}\text{,}\) we use the notation \(\bigcup\limits_{j=1}^{4}C_j\) to mean the same thing as \(\bigcup\limits_{X \in \mathscr{C}}^{}X\text{.}\)

1.

Determine \(\bigcup\limits_{j=1}^{4}C_j\) and \(\bigcap\limits_{j=1}^{4}C_j\text{.}\)

We can see that with the use of subscripts, we do not even have to define the family of sets \(\mathscr{A}\text{.}\) We can work with the infinite family of sets

\begin{equation*} \mathscr{C}^* = \left\{ A_n \mid n \in \N \right\} \end{equation*}

and use the subscripts to indicate which sets to use in a union or an intersection.

2.

Use the roster method to specify each of the following pairs of sets. The universal set is \(\N\text{.}\)

(a)

\(\bigcup\limits_{j=1}^{6}C_j\) and \(\bigcap\limits_{j=1}^{6}C_j\)

(b)

\(\bigcup\limits_{j=1}^{8}C_j\) and \(\bigcap\limits_{j=1}^{8}C_j\)

(c)

\(\bigcup\limits_{j=4}^{8}C_j\) and \(\bigcap\limits_{j=4}^{8}C_j\)

(d)

\(\left( \bigcap\limits_{j=1}^{4}C_j \right)^c\) and \(\bigcup\limits_{j=1}^{4}C_j^c\)
